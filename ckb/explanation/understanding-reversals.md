# تێگەیشتن لە پێچەوانەکردنەوەکان: چاککردنەوەی هەڵە داراییەکان

ئەم ڕێبەرە ڕوونی دەکاتەوە چۆن هەڵەکان لە تۆمارە داراییەکانتدا بە سەلامەتی چاک بکەیتەوە. لە ژمێریاریدا، ئێمە پەیڕەوی یاسایەکی زێڕین دەکەین: **هەرگیز مەسڕەوە، هەمیشە پێچەوانەی بکەرەوە.**

---

## یاسای زێڕین: هەرگیز مەسڕەوە، هەمیشە پێچەوانەی بکەرەوە

لە جیهانێکی نموونەییدا، ئێمە هەڵە ناکەین. بەڵام لە جیهانی ڕاستەقینەدا، هەڵە دەکەین. کاتێک تۆ مامەڵەیەک بە هەڵە تۆمار دەکەیت (وەک بڕی هەڵەی پسووڵەیەک یان پارەدانێک بۆ فرۆشیارێکی هەڵە)، ڕەنگە ویستی ئەوەت هەبێت تەنها دوگمەی "سڕینەوە" (Delete) لێ بدەیت.

**بەڵام لە ژمێریاریدا، سڕینەوە مەترسیدارە چونکە:**
1.  **شوێنپێی وردبینی دەشکێنێت**: تۆ مێژووی ئەوەی ڕوویداوە لەدەست دەدەیت.
2.  **بۆشایی دروست دەکات**: ژمارەکانی پسووڵە دەبێت بەدوای یەکدا بن (INV-001, INV-002, INV-003). سڕینەوەی INV-002 گومان لای وردبینان دروست دەکات.
3.  **زۆرجار نایاساییە**: زۆرێک لە یاساکانی باج سڕینەوەی مامەڵە نێردراوەکان (Posted) قەدەغە دەکەن.

**چارەسەر: پێچەوانەکردنەوە (Reversal)**
لە جیاتی سڕینەوە، ئێمە تۆمارێکی **پێچەوانەکردنەوە** (یان ڕاستکردنەوە) دروست دەکەین. ئەمە مامەڵەیەکی نوێیە کە کاریگەری هەڵەکە "سفر دەکاتەوە".

### وێناکردنی چەمکەکە

وەک بیرکاری بیر لێ بکەرەوە. ئەگەر بە هەڵە +100ت زیاد کرد، تۆ نایسڕیتەوە. تۆ -100 زیاد دەکەیت بۆ ئەوەی بگەڕێیتەوە بۆ سفر.

```
هەڵەی سەرەکی:      + $100.00  (داهاتی هەڵە)
تۆماری پێچەوانە:    - $100.00  (داهاتی نەرێنی)
---------------------------------
ئەنجامی کۆتایی:        $0.00    (چاککراوە)
```

## پێچەوانەکردنەوەی پسووڵەکان (Invoices)

کاتێک کڕیارێک کاڵایەک دەگەڕێنێتەوە یان تۆ بە هەڵە پسووڵەت بۆ کردوون، دەبێت پسووڵەکە پێچەوانە بکەیتەوە.

### بۆ پسووڵەی کڕیار (فرۆشتن)
پێچەوانەکردنەوەی پسووڵەی کڕیار **پسوڵەی گەڕاندنەوە (Credit Note)** دروست دەکات.
1.  بڕۆ بۆ سەر پسووڵە نێردراوەکە (Posted Invoice).
2.  کلیک لە **دروستکردنی پسوڵەی گەڕاندنەوە** (یان "پێچەوانەکردنەوە") بکە.
3.  سیستەمەکە بەڵگەنامەیەکی نوێ (پسوڵەی گەڕاندنەوە) بە بڕی نەرێنی دروست دەکات.
4.  **هاوتاکردن (Reconcile)** بکە لە نێوان پسوڵەی گەڕاندنەوە و پسووڵە ڕەسەنەکە بۆ "پاککردنەوەی".

### بۆ پسووڵەی فرۆشیار (کڕین)
پێچەوانەکردنەوەی پسووڵەی فرۆشیار **پسوولەی وەرگرتنەوە (Debit Note)** دروست دەکات.
1.  بڕۆ بۆ سەر پسووڵەی کڕین (Vendor Bill).
2.  کلیک لە **دروستکردنی پسوولەی وەرگرتنەوە** بکە.
3.  ئەمە ئەو بڕە پارەیە کەم دەکاتەوە کە قەرزاری فرۆشیارەکەیت.

---

## پێچەوانەکردنەوەی پارەدانەکان (Payments)

هەندێک جار تۆ پارەدانێک بە هەڵە تۆمار دەکەیت—ڕەنگە ڕێکەوتی هەڵە، یان جۆرناڵی هەڵەت هەڵبژاردبێت، یان چەکەکە گەڕابێتەوە.

### چۆنێتی پێچەوانەکردنەوەی پارەدانێک
1.  بڕۆ بۆ **ژمێریاری ← پارەدانەکان**.
2.  پارەدانە پشتڕاستکراوەکە (Confirmed Payment) بکەرەوە.
3.  کلیک لە **هەڵوەشاندنەوە (Cancel)** (یان پێچەوانەکردنەوە) بکە.
    *   *تێبینی: ئەگەر پارەدانەکە پێشتر لەگەڵ ڕاپۆرتی بانکدا هاوتا کرابێت (Reconciled)، دەبێت سەرەتا **هاوتانەکردن (Unreconcile)** بکەیت.*
4.  سیستەمەکە دۆخەکەی دەگۆڕێت بۆ **هەڵوەشاوە (Cancelled)** و تۆمارێکی ژمێریاری پێچەوانە دەنێرێت بۆ چاککردنەوەی دەفتەرەکەت.

### چی ڕوودەدات لە پشتەوە؟
ئەگەر لە سەرەتادا تۆمارت کردبێت:
```
قەرزدار (Dr.): بانک            $500
    قەرزدەر (Cr.): هەژماری وەرگیراوەکان  $500
```

پێچەوانەکردنەوەکە تۆماری دەکات:
```
قەرزدار (Dr.): هەژماری وەرگیراوەکان  $500
    قەرزدەر (Cr.): بانک                 $500
```
ئەمە پارەکە دەخاتەوە "ناو" قەرزی کڕیارەکە (کەواتە هێشتا قەرزاری تۆن) و لە باڵانسی بانکتی کەم دەکاتەوە لە سیستەمەکەدا.

---

## هاوتانەکردن: لێکردنەوەی پەیوەندی پارەدان

هەندێک جار پارەدانەکە خۆی ڕاستە، بەڵام بە هەڵە لەگەڵ پسووڵەیەکدا هاوتا کراوە. پێویست ناکات خودی *پارەدانەکە* پێچەوانە بکەیتەوە، تەنها *هاوتاکردنەکە*.

1.  بڕۆ بۆ پارەدانەکە یان پسووڵەکە.
2.  بگەڕێ بۆ بەشی **تەرخانکردنی پارە** (Payment Allocation) یان **باڵانسەکان**.
3.  کلیک لە **هاوتانەکردن (Unreconcile)** یان **لابردنی تەرخانکردن** بکە.

**ئەنجام:**
*   پسووڵەکە دەبێتەوە **کراوە (Open)** (پێنەدراو).
*   پارەدانەکە دەبێتەوە **تەرخاننەکراو (Unallocated)** (لە هەژماری کڕیارەکە دەمێنێتەوە، ئامادەیە بۆ ئەوەی دیسان بەکاربهێنرێتەوە).
*   هیچ پارەیەک "نەجووڵاوە"، تەنها پەیوەندی نێوان بەڵگەنامەکان پچڕا.

---

## چارەسەرکردنی کێشەکان

### پرسیار: بۆچی دوگمەی "پێچەوانەکردنەوە" یان "هەڵوەشاندنەوە" ناچالاکە (ڕەساسیە)؟
ئەمە بەزۆری بە دوو هۆکار ڕوودەدات:

1.  **ڕێکەوتی داخستن (Lock Dates)**: مامەڵەکە هی ماوەیەکە کە داخراوە. تۆ ناتوانیت مێژوو بگۆڕیت لە ساڵ/مانگێکی داخراودا. دەبێت داوا لە بەڕێوەبەر بکەیت ماوەکە بکاتەوە یان چاککردنەوەکە لە ماوەی کراوەی *ئێستادا* ئەنجام بدەیت.
2.  **هاوتاکراو (Reconciled)**: تۆ ناتوانیت پارەدانێک هەڵبوەشێنیتەوە کە لەگەڵ ڕاپۆرتی بانکدا هاوتا کرابێت. دەبێت سەرەتا بڕۆیت بۆ هاوتاکردنی بانکی و پەیوەندییەکە لابدەیت.

### پرسیار: ئایا دەتوانم تەنها پسووڵەکە دەستکاری بکەم؟
ئەگەر پسووڵەکە لە دۆخی **ڕەشنووس (Draft)** بێت، بەڵێ! دەتوانیت دەستکاری بکەیت، بیسڕیتەوە، یان هەر شتێک بگۆڕیت.
کاتێک **نێردرا (Posted)**، ئەوا یاسایی و کۆتاییە. دەبێت ڕێگای پێچەوانەکردنەوە بەکاربهێنیت.
