# تێگەیشتن لە پێشکەشکردنی نرخ (Sales Quotes)

ئەم ڕێبەرییە ڕوونی دەکاتەوە چۆن لیستی نرخ (Quotes) بەکاربهێنیت بۆ پێشنیارکردنی نرخ و بەرهەمەکان بۆ کڕیارەکانت پێش ئەوەی فرۆشتنەکە ئەنجام بدەیت. ئەمە بۆ تیمەکانی فرۆشتن و خاوەن کارەکان نووسراوە، کە باس لە دروستکردن، ناردن بۆ کڕیار، و گۆڕینی بۆ داواکاری دەکات.

---

## پێشکەشکردنی نرخ (Sales Quote) چییە؟

بیری لێ بکەرەوە وەک "مینیۆ"یەکی تایبەت کە بۆ کڕیارێکی دیاریکراو دروستی دەکەیت. ئەو کاڵایانەی دەوێت لەگەڵ ئەو نرخەی تۆ پێشنیاری دەکەیت تێیدایە، بەڵام هێشتا پسووڵەی کۆتایی نییە—بەڵکو بانگهێشتێکە بۆ ئەنجامدانی بازرگانی.

**سێ لایەنە سەرەکییەکە:**
1.  **تۆ (فرۆشیار)**: نرخەکە دروست دەکەیت بە باشترین پێشنیاری خۆت.
2.  **کڕیار**: پێداچوونەوە دەکات و بڕیار دەدات ئایا پەسەندی دەکات یان نا.
3.  **سیستەم**: چاودێری ڕێکەوتی بەسەرچوون و دۆخەکەی دەکات تا لەبیرت نەچێت بەدواداچوونی بۆ بکەیت.

**چرا ئەمە گرنگە؟**
1.  **پیشەیی بوون**: ناردنی نرخێکی فەرمی باشتر دەردەکەوێت لە ئیمەیڵێکی خێرا یان پەیوەندییەکی تەلەفۆنی.
2.  **وردی**: دڵنیایی دەدات کە هەم تۆ و هەم کڕیار لەسەر ئەوە ڕێککەوتوون کە *چی* دەفرۆشرێت و بە *چەند*.
3.  **ڕێکخستنی کار**: بە ئاسانی دەگۆڕدرێت بۆ داواکاری فرۆشتن (Sales Order) یان پسووڵە (Invoice) بەبێ ئەوەی هەموو شتێک بنووسیتەوە.

---

## چۆن لە سیستەمی Kezi کار دەکات

### 1. دۆخەکانی نرخ (Statuses)

هەموو نرخێک بە سووڕێکی ژیانی دیاریکراودا تێدەپەڕێت بۆ ئەوەی یارمەتیت بدات بەدواداچوون بۆ بەرەوپێشچوونەکەی بکەیت.

| دۆخ | ڕەنگ | وەسف |
| :--- | :--- | :--- |
| **ڕەشنووس (Draft)** | خۆڵەمێشی | هێشتا کاری لەسەر دەکەیت. بۆ کڕیار نەنێردراوە. |
| **نێردراو (Sent)** | شین | بۆ کڕیارت ناردووە. ئێستا چاوەڕێی وەڵامیت. |
| **پەسەندکراو (Accepted)** | سەوز | هەواڵێکی خۆش! کڕیارەکە وتی "بەڵێ". |
| **ڕەتکراوە (Rejected)** | سوور | کڕیارەکە وتی "نەخێر". |
| **بەسەرچوو (Expired)** | پرتەقاڵی | ڕێکەوتی کارکردنی تێپەڕیوە بەبێ وەڵام. |
| **گۆڕدراو (Converted)** | سەرەکی | نرخە پەسەندکراوەکە بووە بە داواکاری فرۆشتن یان پسووڵە. |
| **هەڵوەشێنراوە (Cancelled)** | خۆڵەمێشی | بڕیارت دا ئەم نرخە بە دەستی بوەستێنیت. |

### 2. ماوەی کارکردن (Validity Periods)

نرخەکان زۆرجار ڕێکەوتی بەسەرچوونیان هەیە (بۆ نموونە، "بۆ ماوەی 30 ڕۆژ کاری پێ دەکرێت"). ئەمە پارێزگاریت لێ دەکات لە گۆڕانکاری نرخ.
*   سیستەمەکە چاودێری ڕێکەوتی **کاری پێ دەکرێت تا (Valid Until)** دەکات.
*   ئەگەر ئەو ڕێکەوتە تێپەڕێت، سیستەمەکە بە **بەسەرچوو** دایدەنێت.
*   **باشترین پەیڕەو**: هەمیشە ڕێکەوتی بەسەرچوون دابنێ بۆ دروستکردنی پەلە!

## هەنگاو بە هەنگاوی کارکردن

### هەنگاوی 1: دروستکردنی نرخ 📝

بڕۆ بۆ: **فرۆشتن ← نرخەکان ← دروستکردنی نرخ**

1.  **کڕیار**: ئەو کڕیارە هەڵبژێرە کە داوای نرخەکەی کردووە.
2.  **ڕێکەوتی نرخ**: ڕێکەوتی ئەمڕۆ بە ئۆتۆماتیکی دادەنرێت.
3.  **کاری پێ دەکرێت تا**: ڕێکەوتێک دابنێ (بۆ نموونە، 2 هەفتە لە ئێستاوە) بۆ کاتی کۆتایی هاتنی ئۆفەرەکە.

### هەنگاوی 2: زیادکردنی بەرهەم/خزمەتگوزاری

لە بەشی **هێڵەکان (Lines)**، ئەوە زیاد بکە کە دەیفرۆشیت:
1.  **بەرهەم** هەڵبژێرە.
2.  سەیری **بڕ (Quantity)** بکە.
3.  دڵنیابەرەوە لە **نرخی یەکە** (لە لیستی بەرهەمەوە دەیهێنێت، بەڵام دەتوانیت دەستکاری بکەیت).
4.  **داشکاندن** زیاد بکە ئەگەر دەتەوێت گرێبەستێکی تایبەت بکەیت.

### هەنگاوی 3: ناردن بۆ کڕیار ✉️

کاتێک لە ڕەشنووسەکە ڕازی بوویت:
1.  کلیك لەسەر **دیاریکردن وەک نێردراو** بکە.
2.  ئەمە دۆخەکە دەگۆڕێت بۆ **نێردراو (Sent)**.
3.  ئێستا دەتوانیت PDFـەکە چاپ بکەیت یان بە ئیمەیڵ بۆ کڕیاری بنێریت.

### هەنگاوی 4: پشتڕاستکردنەوەی کڕیار ✅

کاتێک کڕیار وەڵام دەداتەوە:
*   **ئەگەر پەسەندی کرد**: کلیك لەسەر **دیاریکردن وەک پەسەندکراو** بکە.
*   **ئەگەر ڕەتی کردەوە**: کلیك لەسەر **دیاریکردن وەک ڕەتکراوە** بکە. دەتوانیت هۆکارێک زیاد بکەیت (بۆ نموونە، "نرخ زۆر بەرزە") بۆ شیکاری داهاتوو.

### هەنگاوی 5: گۆڕین بۆ داواکاری فرۆشتن 🔄

دوای پەسەندکردن، پێویست ناکات داتاکان بنووسیتەوە.
1.  کلیك لەسەر **گۆڕین بۆ داواکاری فرۆشتن** بکە.
2.  سیستەمەکە **داواکارییەکی فرۆشتن (Sales Order)**ی نوێ دروست دەکات بە هەمان کاڵا و نرخەکانەوە.
3.  دۆخی نرخەکە دەگۆڕێت بۆ **گۆڕدراو (Converted)**.

## وێناکردنی سووڕی ژیانی نرخ

```
    [ڕەشنووس] 
       │
       ▼
     [نێردراو] ───┐
       │         │
       ▼         ▼
 [پەسەندکراو] [ڕەتکراوە]
    │    │
    │    └─ [بەسەرچوو]
    ▼
[گۆڕدراو]
```

## باشترین پەیڕەوەکان

1.  **ڕێکەوتی بەسەرچوونی واقیعی دابنێ**: نرخەکان بۆ هەتایە بە کراوەیی مەهێڵەرەوە. نرخەکان دەگۆڕێن! 15-30 ڕۆژ ستانداردە.
2.  **تێبینی بەکاربهێنە**: تێبینییەکی کەسی لە خانەی "تێبینییەکان" زیاد بکە (بۆ نموونە، "بەستنی بێبەرامبەر لەخۆ دەگرێت") بۆ زیادکردنی بەها.
3.  **بەدواداچوون**: هەفتانە سەیری نرخە **نێردراوەکانت** بکە. ئەگەر یەکێکیان خەریکە بەسەردەچێت، بیرخستنەوەیەکی دۆستانە بۆ کڕیار بنێرە.
4.  **چاودێری ڕەتکردنەوەکان**: ئەگەر کڕیارێک نرخێکی ڕەتکردەوە، وەک "ڕەتکراوە" دیاری بکە نەک تەنها بیسڕیتەوە. ئەمە یارمەتیت دەدات دواتر لە ڕێژەی سەرکەوتن/دۆڕان تێبگەیت.

## چارەسەرکردنی پرسیارە باوەکان

**پرسیار: بۆچی ناتوانم دەستکاری نرخێکی نێردراو بکەم؟**
وەڵام: کاتێک نرخێک **نێردرا**، دادەخرێت بۆ دڵنیابوون لەوەی ئەوەی تۆ دەیبینیت لەگەڵ ئەوەی کڕیار هەیەتی یەک دەگرێتەوە. ئەگەر پێویستت بە گۆڕانکارییە، **پێداچوونەوە (Revision)**یەک دروست بکە یان بیگەڕێنەرەوە بۆ ڕەشنووس (ئەگەر ڕێگەپێدراو بێت).

**پرسیار: چی بەسەر نرخە بەسەرچووەکان دێت؟**
وەڵام: لە سیستەمەکەدا دەمێننەوە بەڵام وەک **بەسەرچوو** دیاری دەکرێن. هێشتا دەتوانیت بیانکەیتەوە و نرخێکی نوێ و نوێکراوە پێشنیار بکەیت ئەگەر کڕیارەکە دواتر گەڕایەوە.

**پرسیار: ئایا دەتوانم داواکاری فرۆشتن تێپەڕێنم و ڕاستەوخۆ بچم بۆ پسووڵە؟**
وەڵام: بەڵێ! ئەگەر پرۆسەیەکی سادەت هەیە، دەتوانیت کردارەکەی **گۆڕین بۆ پسووڵە (Convert to Invoice)** ڕاستەوخۆ لە نرخێکی پەسەندکراوەوە بەکاربهێنیت.
