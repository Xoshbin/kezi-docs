# تێگەیشتن لە یەکخستنەوە (Reconciliation)

یەکخستنەوە بریتییە لە پرۆسەی دڵنیابوونەوە لەوەی کە دوو کۆمەڵە تۆمار (بەزۆری باڵانسی دوو ئەژمێر) لەگەڵ یەکتر دەگونجێن. لە جمیار، ئەمە بە پلەی یەکەم ئاماژەیە بۆ **یەکخستنەوەی بانکی**: گونجاندنی تۆمارە ژمێریارییە ناوخۆیییەکانت لەگەڵ بەیاننامە ڕاستەقینەکانی بانکت.

## بۆچی یەکخستنەوە ئەنجام دەدەین؟

یەکخستنەوە کۆنترۆڵێکی ناوخۆیی گرنگە کە دڵنیایی دەدات لەوەی تۆمارەکانت ورد و تەواون. یارمەتیدەرت دەبێت بۆ:
- **دڵنیابوونەوە لە وردی**: دڵنیابوون لەوەی هەر مامەڵەیەک لەسەر بەیاننامەی بانکت هەیە، لە کتێبەکانتدا تۆمار کراوە.
- **دۆزینەوەی هەڵەکان**: دۆزینەوەی ئەو هەڵانەی کە بانکەکە یان خۆت لە کاتی داخڵکردنی داتاکاندا کردووتە.
- **ڕێگری لە ساختەکاری**: ناسینەوەی ئەو مامەڵانەی کە ڕێگەپێنەدراون.
- **چاودێریکردنی نه‌ختینە (Cash Flow)**: زانینی بڕی وردی ئەو نەختینەیەی کە بەردەستتە.

## پرۆسەی یەکخستنەوە

جمیار سیستمێکی دوو قۆناغی "چاوەڕوانکراو" (Outstanding) بەکاردێنێت بۆ پاراستنی یەکپارچەیی تۆماری دووسەرە. ئەمە دڵنیایی دەدات لەوەی کە قەرزەکە پاکتاو دەکرێت لە کاتی نوسینی چەکەکەدا، بەڵام نەختینەکە تەنها ئەو کاتە کەم دەبێتەوە کە بانکەکە بە ڕاستی چەکەکە خەرج دەکات.

### ١. تۆمارکردنی پارەدانی ناوخۆیی
کاتێک پارە بە دابینکەرێک دەدەیت یان پارە لە کڕیارێک وەردەگریت، **پارەدانێک (Payment)** تۆمار دەکەی.
- **کاریگەری دارایی**: باڵانسەکە لە ئەژمێری هاوبەش (A/P یان A/R) دەگوازرێتەوە بۆ **ئەژمێرێکی چاوەڕوانکراو (Outstanding Account)**.
- **بارودۆخ**: `Confirmed` (پەسەندکراو).
- **نموونەی تۆماری ڕۆژنامە (پارەدان بە دابینکەر)**:
  - قەرزدار (Debit): ئەژمێری پارەپێدراوەکان (Accounts Payable)
  - خاوەنقەرز (Credit): **پارەدانە چاوەڕوانکراوەکان (Outstanding Payments)**

### ٢. گونجاندن لەگەڵ بەیاننامەی بانک
کاتێک بەیاننامەی بانک وەردەگریت، ئامرازی **گونجێنەری یەکخستنەوە (Reconciliation Matcher)** بەکاردێنیت بۆ بەستنەوەی بەو پارەدانەی کە پەسەند کراوە.
- **کاریگەری دارایی**: باڵانسەکە لە ئەژمێری چاوەڕوانکراوەوە دەگوازرێتەوە بۆ **ئەژمێری بانکی** ڕاستەقینە.
- **بارودۆخ**: `Reconciled` (یەکخراوەتەوە).
- **نموونەی تۆماری ڕۆژنامە**:
  - قەرزدار (Debit): **پارەدانە چاوەڕوانکراوەکان (Outstanding Payments)**
  - خاوەنقەرز (Credit): ئەژمێری بانکی

## یەکخستنەوەی ڕاستەوخۆ (Write-offs)

هەندێک جار مامەڵەیەکی بانکی هەیە کە پارەدانی ناوخۆیی پێشوەختەی بۆ نەکراوە (وەک کرێی بانک یان سوود). لەم حاڵەتانەدا، دەتوانیت **یەکخستنەوەی ڕاستەوخۆ** ئەنجام بدەیت.
- واتە "سڕینەوەی" هێڵی بەیاننامەکە ڕاستەوخۆ بۆ ئەژمێری خەرجی یان داهات.
- **نموونەی تۆماری ڕۆژنامە (کرێی بانک)**:
  - قەرزدار (Debit): خەرجی کرێی بانکی
  - خاوەنقەرز (Credit): ئەژمێری بانکی

## چەمکە سەرەکییەکان

| چەمک | وەسف |
| :--- | :--- |
| **ئەژمێری چاوەڕوانکراو** | ئەژمێرێکی کاتییە کە بۆ ڕاگرتنی باڵانس بەکاردێت تاوەکو لەلایەن بانکەوە پشتڕاست دەکرێتەوە. |
| **هێڵی بەیاننامە** | مامەڵەیەکی تاکی ناو بەیاننامەی بانکی. |
| **گونجێنەر (Matcher)** | ڕوکارە بینراوەکە کە بۆ بەستنەوەی هێڵەکانی بەیاننامە بە پارەدانە ناوخۆیییەکان بەکاردێت. |
| **یەکخراوەتەوە (Reconciled)** | دۆخی کۆتایی مامەڵەکە کە تێیدا تۆمارە ناوخۆیییەکان و تۆمارەکانی بانک یەکانگیر دەبن. |
