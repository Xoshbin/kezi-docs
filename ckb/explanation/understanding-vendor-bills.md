پسووڵەی فرۆشیار بریتییە لە تۆمارێکی دیجیتاڵی بۆ شکڵێک لە پسووڵەی بازرگانی (Invoice) کە لە دابینکەرێکەوە وەرگیراوە. لە چوارچێوەی Kezi  ـدا، ئەمە تەنها داوایەک نییە بۆ پارەدان، بەڵکو بەڵگەنامەیەکی سەرەکییە کە دەبێتە هۆی تۆمارکردنی خەرجییەکان، نوێکردنەوەی تێچووی کۆگا، و جێگیرکردنی سامانی جێگیر (Capitalization).

ئەم ڕێبەرە چوارچێوەی فیکری پسووڵەکانی فرۆشیار، خولی ژیان، و ڕۆڵیان لە پرۆسەی ژمێریاریی کەڵەکەبوو (Accrual Accounting) ڕوون دەکاتەوە.

---

## ڕۆڵی پسووڵەکانی فرۆشیار

پسووڵەی فرۆشیار چوار فرمان/ئەرکی سەرەکی لە خولی ژمێریاریدا ئەنجام دەدات:

1.  **جێگیرکردنی قەرز (Liability Establishment)**: بە فەرمی دان بەوەدا دەنێت کە کۆمپانیاکە پارەی قەرزداری لایەنی سێیەمە (Accounts Payable).
2.  **تۆمارکردنی خەرجی (Expense Recognition)**: خەرجییەکە لەو ماوە دیاریکراوەدا تۆمار دەکات کە تێیدا ئەنجامدراوە، بەبێ گوێدان بەوەی کەی پارەکە دەدرێت (بەپێی بنەمای کەڵەکەبوون - Accrual Principle).
3.  **بەهاپێدانی کۆگا (Inventory Valuation)**: بۆ کاڵاکانی کۆگا، پسووڵەکە تێچووی کۆتایی ئەو بڕگانە دیاری دەکات کە وەرگیراون، کە ڕەنگە جیاواز بێت لەو نرخەی لە سەرەتادا لە داواکاری کڕین (Purchase Order) دیاری کراوە.
4.  **وەرگرتنەوەی باج (Tax Reclamation)**: وەک بەڵگەنامەیەک کار دەکات بۆ وەرگرتنەوەی باجی ناوخۆ (Input VAT) یان باجە گەڕاوەکانی تر.

### پسووڵەی فرۆشیار بەرامبەر داواکاری کڕین (Purchase Order)

زۆربەی کات ئەم دوو بەڵگەنامەیە تێکەڵ دەکرێن. **داواکاری کڕین (PO)** بریتییە لە *داواکردنێک* بۆ کڕینی کاڵا، کە وەک گرێبەستێکە و بۆ فرۆشیار دەنێردرێت. **پسووڵەی فرۆشیار (Vendor Bill)** بریتییە لەو داوای پارەدانەی کە لە فرۆشیارەوە دێت.

| تایبەتمەندی | داواکاری کڕین | پسووڵەی فرۆشیار |
| :--- | :--- | :--- |
| **ئاراستە** | دەرەوە (بۆ فرۆشیار) | ناوەوە (لە فرۆشیارەوە) |
| **کاریگەری ژمێریاری** | نییە (بەزۆری) | پارەی قەرزی فرۆشیاران زیاد دەکات |
| **مەبەست** | ڕێپێدان بۆ کڕین | پابەندبوون بۆ پارەدان |
| **کات** | پێش گەیاندن | دوای گەیاندنی کاڵا یان خزمەتگوزاری |

## خولی ژیانی پسووڵە

پسووڵەی فرۆشیار بە زنجیرەیەک قۆناغی دیاریکراودا تێدەپەڕێت بۆ دڵنیابوون لە دروستیی داتاکان.

### 1. ڕەشنووس (Draft)
قۆناغی سەرەتاییە. پسووڵەی ڕەشنووس وەک شوێنکارێکە. هیچ تۆمارێکی ژمێریاری دروست ناکات، و دەتوانرێت بە ئازادی دەستکاری بکرێت یان بسڕدرێتەوە. لێرەدا زانیارییەکان داخل دەکرێن و پشکنینی دروستی بۆ دەکرێت.

### 2. نێردراو (Posted)
پسووڵەی نێردراو تۆمارێکی ژمێریاری نەگۆڕە. گۆڕینی پسووڵەیەک بۆ بارودۆخی "Posted":
*   تۆماری ڕۆژانە (Journal Entry) دروست دەکات.
*   بوارەکان قوفڵ دەکات بۆ ڕێگری لە دەستکاری کردن.
*   باڵانسی فرۆشیار نوێ دەکاتەوە.
*   پەرتووکی گشتی (General Ledger) نوێ دەکاتەوە.

بۆ ڕاستکردنەوەی پسووڵەیەکی Posted، ناتوانیت تەنها دەستکاری بکەیت؛ دەبێت **Debit Note** (Credit Note لە چاوی فرۆشیارەوە) دەربکەیت یان هەڵیوەشێنیتەوە بۆ پێچەوانەکردنەوەی تۆمارەکە.

### 3. دراو (Paid) یان بەشێکی دراو
کاتێک پارە بە پسووڵەکە دەدرێت، بارودۆخەکەی نوێ دەبێتەوە بۆ نیشاندانی باڵانسی ماوە:
*   **بەشێکی دراو (Partially Paid)**: پسووڵەکە لەگەڵ بڕە پارەیەکدا هاوتا کراوە کە کەمترە لە کۆی گشتی پارەکە.
*   **دراو (Paid)**: پسووڵەکە بە تەواوی لەگەڵ یەک یان چەند پارەدانێکدا یەکسان کراوە.

### 4. هەڵوەشاوە (Cancelled)
پسووڵەی هەڵوەشاوە بە تەواوی پووچەڵ کراوەتەوە. سیستەمەکە هەر تۆمارێکی ڕۆژانە کە پەیوەندی پێوە هەبێت پێچەوانە دەکاتەوە، بۆ ئەوەی دڵنیابێتەوە کە کاریگەرییە ژمێریارییەکەی سفرە.

## کاریگەری ژمێریاری

کاتێک پسووڵەیەکی فرۆشیار **Posted** دەکرێت، سیستەمەکە بە شێوەیەکی ئۆتۆماتیکی ڕۆژنامەی دوو-تۆمار دروست دەکات. تێگەیشتن لەم تۆمارە گرنگە بۆ پشکنینی ڕاپۆرتە داراییەکانت.

### پسووڵەی خەرجی ئاسایی
بۆ کڕینی سادەی خزمەتگوزاری یان کاڵای بەکاربەر:

| هەژمار | قەرزدار (Debit) | قەرزدەر (Credit) |
| :--- | :--- | :--- |
| **هەژماری خەرجی** (وەک پێداویستی ئۆفیس) | زیادکردن (Dr) | |
| **هەژماری باج** (Input VAT) | زیادکردن (Dr) | |
| **پارەی قەرزی فرۆشیاران (Accounts Payable)** | | زیادکردن (Cr) |

### پسووڵەی کڕینی سامان (Asset Acquisition Bill)
بۆ کڕینی داراییە جێگیرەکان (وەک لاپتۆپ، ئامێر):

| هەژمار | قەرزدار (Debit) | قەرزدەر (Credit) |
| :--- | :--- | :--- |
| **پاکتاوکاری سامانی جێگیر** / **هەژماری سامان** | زیادکردن (Dr) | |
| **هەژماری باج** (Input VAT) | زیادکردن (Dr) | |
| **پارەی قەرزی فرۆشیاران (Accounts Payable)** | | زیادکردن (Cr) |

## کاتێگۆرییەکانی کڕین

سیستەمەکە بە شێوازی جیاواز مامەڵە لەگەڵ پسووڵەکان دەکات بەپێی ئەوەی چی دەکڕدرێت:

### 1. کڕینی کۆگا
پسووڵەی ئەو کاڵایانەی بۆ کۆگا دەبن کاریگەری لەسەر **بەهاپێدانی کۆگا (Stock Valuation)** هەیە. ئەگەر بەهای کاتیی ڕاستەقینە بەکاربهێنیت، ڕەنگە پسووڵەکە قەرزدار بۆ هەژماری "ناوبژیوانی کۆگا" بکات لەجیاتی خەرجی ڕاستەوخۆ، هەتا کاڵاکە وەردەگیرێت بۆ هاوسەنگکردنی مامەڵەکە.

### 2. خزمەتگوزاری و بەکاربەرەکان
ئەمانە خەرجی ڕاستەوخۆن. یەکسەر کاریگەری لەسەر ڕاپۆرتی قازانج و زیان (P&L) دادەنێن دوای Post کردنی پسووڵەکە.

### 3. خەرجییە سەرمایەییەکان (CapEx)
ئەم پسووڵانە یەکسەر ناچنە ڕاپۆرتی قازانج و زیان. لە جیاتی ئەوە، **سامانێک (Asset)** لە مەیزانییەدا دروست دەکەن. خەرجییەکە بە تێپەڕبوونی کات لە ڕێگەی *داخوران (Depreciation)* ـەوە تۆمار دەکرێت.

## زنجیرەی پەسەندکردنی ئۆتۆماتیکی

بۆ ڕێگری لە فێڵ و فیشەڵ و پاراستنی کۆنتڕۆڵی بودجە، زۆربەی کات پسووڵەی فرۆشیار پێویستی بە پەسەندکردن (Approval) هەیە پێش ئەوەی بتوانرێت Post بکرێت. سیستەمەکە پشتگیری لە سنووری جیاواز دەکات بۆ پەسەندکردن بەپێی بڕی پسووڵەکە:

*   **ئاستی بەکارهێنەر**: دەتوانێت ڕەشنووس دروست بکات بەڵام پێویستی بە بەڕێوەبەر هەیە بۆ Post کردن.
*   **ئاستی بەڕێوەبەر**: دەتوانێت پسووڵەکان پەسەند بکات تا سنوورێکی دیاریکراو (بۆ نموونە 5,000,000 دینار).
*   **ئاستی بەڕێوەبەری گشتی یان دارایی**: پێویستە بۆ ئەو بڕانەی لە سنوورە ستانداردەکان تێپەڕ دەبن.

ئەم پۆلێنکردنی ئەرکانە (**Segregation of Duties**) دڵنیایی دەدات کە ئەو کەسەی داوای کڕینەکە دەکات هەمان ئەو کەسە نییە کە ڕێگە بە پارەدانەکە دەدات.
