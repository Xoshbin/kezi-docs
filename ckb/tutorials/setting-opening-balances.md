# وانەی فێرکاری: دانانی دەستپێکردنی یەکەی هەژمارەکان

بەخێر بێیت بۆ ژمێریار! یەکێک لە هەنگاوە هەرە گرنگەکان لە دەستپێکردن بە سیستەمێکی نوێی ژمێریاری، گواستنەوەی زانیارییە داراییە بەردەستەکانتە. ئەم وانەیە ڕێبەرایەتیت دەکات لە پرۆسەی دانانی **دەستپێکردنی یەکەی هەژمارەکان**—کە وێنەیەکی دارایی بازرگانییەکەتە لەو ڕۆژەی دەست دەکەیت بە بەکارهێنانی ژمێریار.

## چیرۆکەکە: دوکانی مۆبیلیاتی ئەحمەد

ئەحمەد بۆ ماوەی سێ ساڵە دوکانی مۆبیلیاتی هەیە و تۆمارەکانی لە خشتەیەکی ئەلیکترۆنی (Spreadsheet) پاراستووە. ئەمڕۆ دەچێتە سەر سیستەمی ژمێریار. خشتەکەی ئەمانە نیشان دەدات:
- **کاش لە بانک**: $15,000
- **کۆگا (مەخزەن)**: $25,000 (کۆگای کورسی و مێز)
- **قەرز لای کڕیاران**: $5,000
- **قەرز بۆ دابینکەران**: $3,000
- **سەرمایەی سەرەتایی**: $42,000

ئەحمەد دەیەوێت دڵنیا بێتەوە کە "ڕۆژی یەکەم" لە ژمێریاردا لەگەڵ "ڕۆژی کۆتایی" لە ئێکسڵدا (Excel) یەکدەگرێتەوە.

---

## پێشمەرجەکان

پێش ئەوەی دەست پێ بکەین، دڵنیا بەرەوە کە:
1.  **نەخشەی هەژمارەکانت** دروست کردووە (بۆ نموونە: بانک، کۆگا، هەژماری پارەدانەکان).
2.  بەرواری دەستپێکی **ساڵی دارایی** خۆت دیاری کردووە.
3.  هاوسەنگی پێداچوونەوەی (Trial Balance) کۆتایی سیستەمە کۆنەکەت کۆکردووەتەوە.

---

## هەنگاوی ۱: دروستکردنی تۆماری ڕۆژانەی دەستپێک

لە ژمێریاردا، ئێمە **تۆماری ڕۆژانەی دەستی (Manual Journal Entry)** بەکاردەهێنین بۆ تۆمارکردنی خاڵی دەستپێک.

1.  بڕۆ بۆ **ژمێریاری > تۆماری ڕۆژانە**.
2.  کلیک لەسەر **دروستکردنی نوێ** بکە.
3.  **بەروار** دابنێ بۆ ڕۆژێک *پێش* بەرواری فەرمی دەستپێکردنت لە ژمێریاردا (بۆ نموونە: ۳۱ی ۱۲ ئەگەر لە ۱ی ۱ دەست پێ دەکەیت).
4.  **ژمارەی بەڵگە** دابنێ بۆ `Opening Balances 2024`.

---

## هەنگاوی ۲: تۆمارکردنی سامانەکان (Debits)

سامانەکان ئەو شتانەن کە خاوەندارێتییان دەکەیت. لە ژمێریاریدا، ئەمانە باڵانسی **قەرزدار (Debit)**یان هەیە.

ئەحمەد سامانەکانی داخڵ دەکات:
- **هەژماری بانک**: قەرزدار (Debit) $15,000
- **هەژماری کۆگا**: قەرزدار (Debit) $25,000
- **هەژمارەکانی وەرگرتن (لای کڕیار)**: قەرزدار (Debit) $5,000

---

## هەنگاوی ۳: تۆمارکردنی قەرزەکان و خاوەندارێتی (Credits)

قەرزەکان (ئەوەی قەرزاریت) و خاوەندارێتی (پشکی خاوەن) باڵانسی **قەرزدەر (Credit)**یان هەیە.

ئەحمەد قەرزدەرەکانی داخڵ دەکات:
- **هەژماری پارەدانەکان (بۆ دابینکەر)**: قەرزدەر (Credit) $3,000
- **سەرمایەی خاوەن**: قەرزدەر (Credit) $42,000

---

## هەنگاوی ٤: هاوسەنگکردنی تۆمارەکە

بنەمایەکی سەرەکی ژمێریاری دوو-تۆماری ئەوەیە کە **قەرزدارەکان دەبێت یەکسان بن بە قەرزدەرەکان**.

| هەژمار | قەرزدار (Debit) | قەرزدەر (Credit) |
| :--- | :--- | :--- |
| بانک | $15,000 | |
| کۆگا | $25,000 | |
| هەژمارەکانی وەرگرتن | $5,000 | |
| هەژمارەکانی پارەدانەکان | | $3,000 |
| سەرمایەی خاوەن | | $42,000 |
| **کۆی گشتی** | **$45,000** | **$45,000** |

ژمێریار هێمایەکی سەوز (Checkmark) نیشان دەدات کە دەریدەخات تۆمارەکە هاوسەنگە. کلیک لە **ناردن (Post)** بکە.

---

## هەنگاوی ٥: پشتڕاستکردنەوەی هاوسەنگی پێداچوونەوە

ئێستا کە تۆمارەکە نێردراوە، با ئەنجامەکە بپشکنین.

1.  بڕۆ بۆ **ژمێریاری > ڕاپۆرتەکان > هاوسەنگی پێداچوونەوە (Trial Balance)**.
2.  بەرواری دەستپێکردنت هەڵبژێرە.
3.  دڵنیا بەرەوە کە هەر هەژمارێک لەگەڵ خشتەی ئێکسڵەکەتدا یەک دەگرێتەوە.

## کورتە

تۆ بە سەرکەوتوویی "تەندروستی دارایی" بازرگانییەکەتت گواستەوە بۆ ناو ژمێریار! هەموو مامەڵەکانی داهاتوو—فرۆشتن، کڕین، و خەرجییەکان—ئێستا لەسەر ئەم بنەما وردە بنیات دەنرێن.
