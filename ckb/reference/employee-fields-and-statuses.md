# ڕێبەری سەرچاوەی کارمەند

ئەم ڕێبەرە وردەکارییە تەکنیکییەکان سەبارەت بە کێڵگەکان و بەهاکانی بارودۆخ لە مۆدیوڵی کارمەند دابین دەکات.

### کێڵگە سەرەکییەکان

فۆرمی کارمەند دابەشکراوە بۆ چەند بەشێک بۆ ڕێکخستنی داتا ئاڵۆزەکان.

| ناوی کێڵگە | وەسف | مەرجی سەرەکی |
|:---|:---|:---|
| **ناوی تەواو** | ناوی یاسایی کارمەند. | پێویستە |
| **ئیمەیڵ** | ئیمەیڵی فەرمی کۆمپانیا. | بێ هاوتا (Unique) |
| **تەلەفۆن** | ژمارەی پەیوەندی سەرەکی. | پێشنیارکراو |
| **بەش** | ئەو یەکەیەی کە تێیدا کار دەکات. | پێویستە |
| **پلە** | نازناوی فەرمی کارەکەی. | پێویستە |
| **بەڕێوەبەر** | ئەو کەسەی ڕاپۆرتی بۆ دەکات. | بۆ پەسەندکردنەکان |
| **بەرواری دەستبەکاربوون** | یەکەم ڕۆژی فەرمی کارکردن. | پێویستە |
| **بارودۆخ** | دۆخی ئێستای دامەزراندن. | ئۆتۆماتیکی "چالاک"ـە |

---

### وردەکارییە کەسی و دارایییەکان

| ناوی کێڵگە | وەسف |
|:---|:---|
| **بەرواری لەدایکبوون** | بۆ ڕاپۆرتەکانی تەمەن و پابەندبوونی HR بەکار دێت. |
| **ژمارەی ناسنامە / پاسپۆرت** | بۆ ناسنامەی یاسایی لە پەیماننامەکاندا. |
| **ئەژمێری بانکی** | ژمارەی ئەژمێر یان IBAN بۆ گواستنەوەی مووچە. |
| **پەیوەندی کاتی فریاکەوتن** | ناو و تەلەفۆنی کەسێک بۆ کاتی پێویست. |

---

### پێناسەی بارودۆخەکان

کێڵگەی `Status` لەسەر تۆماری کارمەند کۆنترۆڵی دەرکەوتن و بەشدارییان لە پرۆسەکانی سیستمەکەدا دەکات.

| بارودۆخ | ماناکەی | کاریگەری سیستم |
|:---|:---|:---|
| `Active` (چالاک) | دامەزراوە و لە ئێستادا کار دەکات. | لە مووچە، مۆڵەت و ئامادەبووندا هەژمار دەکرێت. |
| `Inactive` (ناچالاک) | نەبوونی کاتی یان پێش دامەزراندن. | لە لیستە چالاکەکاندا دەرناکەوێت. |
| `Terminated` (وازهێنراو) | چیتر فەرمانبەر نییە. | بۆ مێژوو دەمێنێتەوە؛ لە هەموو پڕۆسەکان لادەبرێت. |

---

### هاوپێچەکان

جۆرە باوەکانی بەڵگەنامە کە لە تۆماری کارمەنددا پاشکەوت دەکرێن:

1. **پەیماننامەی واژۆکراو**: کۆپی دیجیتاڵی ڕێککەوتننامەکانی دامەزراندن.
2. **ناسنامە**: سکانی ناسنامە، پاسپۆرت یان مۆڵەتی نیشتەجێبوون.
3. **بەڵگەی خوێندن**: بڕوانامەکانی زانکۆ و خولەکان.
4. **وێنە**: وێنەی فەرمی بۆ ناسنامە یان پرۆفایلی ناو سیستم.
