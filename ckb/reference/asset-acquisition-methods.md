# سەرچاوە: شێوازەکانی بەدەستهێنانی سامان

ئەم سەرچاوەیە بەراوردێکی تەکنیکی لە نێوان شێوازە جیاوازەکانی تۆمارکردنی سامان لە نۆتبووک ئێڵ ئێم (Kezi ) دابین دەکات.

---

## خشتەی بەراوردکاری

| تایبەتمەندی | **پسووڵەی فرۆشیار** (پێشنیارکراو) | **تۆمارکردنی دەستی** | **دیاریکردنی هەژمار** |
| :--- | :--- | :--- | :--- |
| **بەکارهێنانی سەرەکی** | کڕینی سامانی نوێ لە فرۆشیارەکان. | تۆمارکردنی سامانی کۆن/باڵانسە سەرەتاییەکان. | جۆرە سامانە زۆرەکان لە هەمان جۆر. |
| **ئاستی ئۆتۆماتیکی** | **بەرز.** دروستکردنی سامان بە ئۆتۆماتیکی لە کاتی پۆستکردندا. | **نییە.** پێویستی بە داخلکردنی دەستی هەموو زانیارییەکان هەیە. | **مامناوەند.** هەژمارەکان بۆ دروستکردنی سامان ئامادە دەکات. |
| **بەدواداچوونی وردبینی** | بەستراوەتەوە بە فرۆشیار، پسووڵە، و پارەدان. | تۆمارێکی سەربەخۆ. | بەستراوەتەوە بە مامەڵە بنەڕەتییەکە. |
| **سەرچاوەی زانیاری** | نرخ و ڕێکەوت لە پسووڵەکەوە وەردەگرێت. | دەبێت بە دەست داخل بکرێن. | نرخ و ڕێکەوت لە بابەتەکە وەردەگرێت. |

---

## تایبەتمەندییە تەکنیکییەکان

### کێڵگە ناچارییەکان (دۆخی ڕەشنووس)
*   **ناو (Name)**: ناسێنەرێکی ناوازە بۆ سامانەکە.
*   **ناسنامەی کۆمپانیا (Company ID)**: ناسێنەری کۆمپانیا.
*   **ناسنامەی دراو (Currency ID)**: دراوی کڕین.
*   **ڕێکەوتی کڕین (Purchase Date)**: دەبێت لە ماوەیەکی ژمێریاری کراوەدا بێت.

### پێداویستییە ژمێریارییەکان
بۆ پشتڕاستکردنەوەی سامانێک، دەبێت ئەم هەژمارانەی پەرتووکی گشتی (GL) کارا و دروست بن:
١.  **هەژماری سامان**: جۆر = `Fixed Assets`.
٢.  **هەژماری داخورانی کۆکراوە**: جۆر = `Fixed Assets` یان `Contra-Asset`.
٣.  **هەژماری خەرجی داخوران**: جۆر = `Expense`.

### لۆژیکی گۆڕانی دۆخەکان
١.  **ڕەشنووس (Draft)**: دەتوانرێت دەستکاری بکرێت. هیچ ئەژمارکردنێکی داخوران پۆست نەکراوە.
٢.  **پشتڕاستکراوەتەوە (Confirmed)**: کێڵگە داراییە سەرەکییەکان قفڵ دەکرێن. تۆمارە خشتەبەندکراوەکان ئەژمار دەکرێن.
٣.  **لە داخوراندان (Depreciating)**: بە لایەنی کەمەوە یەک تۆماری داخوران پۆست کراوە.
٤.  **بە تەواوی داخوراو (Fully Depreciated)**: بەهای کتێبی یەکسانە بە تێچوی هەڵگرتن.
٥.  **فرۆشراو/لادراو (Sold/Disposed)**: تۆماری ژمێریاری لادان دروست کراوە. سامانەکە لە لیستی چالاک لادەبرێت.
