# سەرچاوە: قاڵبەکانی هاوردەکردنی باڵانسی دەستپێک

ئەگەر هەزاران تۆمارت هەیە بۆ گواستنەوە، داخڵکردنی دەستی گونجاو نییە. ئەم سەرچاوەیە بەکاربێنە بۆ ئامادەکردنی فایلەکانی CSV بۆ هاوردەکردنی باڵانسی دەستپێک بۆ ناو ژمێریار (Kezi).

## پەرتووکی گشتی (نەخشەی هەژمارەکان)

**شوێنی مەبەست**: ژمێریاری > نەخشەی هەژمارەکان > هاوردەکردن (Import)

| ناوی خانە (Field) | پێویستە؟ | جۆر | شرۆڤە |
| :--- | :--- | :--- | :--- |
| `code` | بەڵێ | دەق | کۆدی بێوێنەی هەژمار (بۆ نموونە: `110101`). |
| `name` | بەڵێ | دەق | ناوی هەژمار (بۆ نموونە: `بانکی مەنسوور`). |
| `type` | بەڵێ | لیست | یەکێک لەمانە: `asset`, `liability`, `equity`, `revenue`, `expense`. |
| `opening_balance` | بەڵێ | ژمارە | بڕی سەرەتایی هەژمارەکە. |
| `currency_code` | نەخێر | دەق | کۆدی ئایسۆ (بۆ نموونە: `USD`, `IQD`). |

## باڵانسی دەستپێکی هاوبەشەکان (AR/AP)

**شوێنی مەبەست**: فرۆشتن > پسووڵەکان > هاوردەکردن یان کڕین > پسووڵەکانی دابینکەر > هاوردەکردن

بۆ تۆمارکردنی پسووڵە ماوەکان بە شێوەی تاکەکەسی:

| ناوی خانە (Field) | پێویستە؟ | جۆر | شرۆڤە |
| :--- | :--- | :--- | :--- |
| `partner_name`| بەڵێ | دەق | دەبێت لەگەڵ ناوی هاوبەشێکی بەردەست یەکبگرێتەوە. |
| `date` | بەڵێ | بەروار | بەرواری ئەسڵیی مامەڵەکە. |
| `reference` | بەڵێ | دەق | ژمارەی ئەسڵیی پسووڵەکە. |
| `line_items/description` | بەڵێ | دەق | شرۆڤەیەکی کورت (بۆ نموونە: "Opening Balance"). |
| `line_items/account` | بەڵێ | کۆد | **گرنگ**: کۆدی هەژماری "Opening Balance Counterpart" بەکاربێنە. |
| `line_items/price` | بەڵێ | ژمارە | ئەو بڕە پارەیەی کە ماوە بنێردرێت یان بدرێت. |

## باڵانسی دەستپێکی کۆگا

**شوێنی مەبەست**: کۆگا > ڕێکخستنی مەخزەن (Stock Adjustments) > هاوردەکردن

| ناوی خانە (Field) | پێویستە؟ | جۆر | شرۆڤە |
| :--- | :--- | :--- | :--- |
| `warehouse` | بەڵێ | دەق | ناوی شوێن یان مەخزەنەکە. |
| `product_code`| بەڵێ | دەق | کۆدی بێوێنەی (SKU) بەرهەمەکە. |
| `quantity` | بەڵێ | ژمارە | بڕی بەرهەمی بەردەست لە مەخزەن. |
| `unit_price` | بەڵێ | ژمارە | تێچووی هەر دانەیەک بۆ نرخاندن. |
| `counterpart` | بەڵێ | کۆد | کۆدی هەژماری "Opening Balance Equity" بەکاربێنە. |

---

## یاسا تەکنیکییەکانی فۆرماتکردن

1.  **فۆرماتی بەروار**: `YYYY-MM-DD` بەکاربێنە (بۆ نموونە: `2024-12-31`).
2.  **نیشانەی کۆما**: خاڵ `.` وەک نیشانەی جیاکەرەوەی ژمارە بەکاربێنە (بۆ نموونە: `1500.50`).
3.  **کۆدکردن (Encoding)**: فایلەکان بە **UTF-8 (Comma Delimited)** پاشەکەوت بکە بۆ پاشەکەوتکردنی پیتە کوردییەکان بە دروستی.
4.  **پارە (Currency)**: ئەگەر بە پارەی بیانی هاوردە دەکەیت، دڵنیا بەرەوە نرخی ئاڵوگۆڕ (exchange_rate) دابین کراوە.
