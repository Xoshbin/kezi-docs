# ڕێبەری سەرچاوەی مووچە

ئەم ڕێبەرە وردەکاری تەکنیکی سەبارەت بە قۆناغەکانی کاری مووچە و تۆمارە ژمێریارییە پەیوەندیدارەکان دابین دەکات.

### بارودۆخەکانی پرۆسەی کار (Workflow Statuses)

هەر تۆمارێکی مووچە بە چەند قۆناغێکدا تێدەپەڕێت بۆ دڵنیابوون لە وردبینی و ڕێپێدان.

| بارودۆخ | مانا | کاریگەری سیستم |
|:---|:---|:---|
| **Draft** (ڕەشنووس) | هەژمارکردن لە پرۆسەدایە. | دەتوانرێت دەستکاری بکرێت؛ کاریگەری ژمێریاری نییە. |
| **Processed** (پەسەندکراو) | لەلایەن بەڕێوەبەرەوە پەسەند کراوە. | قفڵ دەکرێت؛ **تۆماری ژمێریاری** دروست دەکات (Dr Expense / Cr Payable). |
| **Paid** (دراوە) | پارەکە گواستراوەتەوە. | دەبەسترێتەوە بە تۆمارێکی **پارەدان**؛ لایەنی قەرز پاکتاو دەکات. |
| **Closed** (داخراو) | کۆتایی پێهاتووە و مێژووییە. | چیتر دەستکاری ناکرێت. |

---

### لۆژیکی ژمێریاری

کاتێک مووچە لە قۆناغی **Draft** دەگۆڕدرێت بۆ **Processed**، سیستمەکە تۆمارێکی ژمێریاری دروست دەکات. لە خوارەوە نموونەیەکی ستاندارد دەبینیت:

**سیناریۆ**: مووچەی مانگانەی 1,000 دۆلار.

| ئەژمێر | تەرەفی مەدین (Debit) | تەرەفی داین (Credit) | تێبینی |
|:---|:---|:---|:---|
| 601001 - خەرجی مووچە | $1,000.00 | | تێچووی کۆمپانیا تۆمار دەکات. |
| 201001 - مووچەی شایستە/قەرز | | $1,000.00 | قەرزی کارمەند تۆمار دەکات. |

**کاتێک پارەکە دەدرێت:**

| ئەژمێر | تەرەفی مەدین (Debit) | تەرەفی داین (Credit) | تێبینی |
|:---|:---|:---|:---|
| 201001 - مووچەی شایستە/قەرز | $1,000.00 | | قەرزەکە پاکتاو دەکات. |
| 101001 - بانک/کاش | | $1,000.00 | دەرچوونی پارە لە بانک تۆمار دەکات. |

---

### سەرچاوەکانی داتا

ئەم کێڵگانەی خوارەوە بە شێوەیەکی ئۆتۆماتیکی لە مۆدیوڵەکانی ترەوە وەردەگیرێن:

*   **ناوی کارمەند/ID**: لە تۆماری کارمەندەوە.
*   **مووچەی بنەڕەتی**: لە پەیماننامەی کارە چالاکەکەوە.
*   **ئەژمێری بانکی**: لە بەشی دارایی تۆماری کارمەندەوە.
*   **بڕینی ئامادەنەبوون**: بەپێی تۆمارەکانی [بەڕێوەبردنی مۆڵەت](../how-to/manage-leave.md) کە بە "بێ مووچە" نیشانەکراون.
