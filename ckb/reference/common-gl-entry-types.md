# جۆرە باوەکانی تۆمارەکانی ڕۆژانە (GL)

ئەم ڕێبەرە خشتەیەکی خێرا پێشکەش دەکات بۆ مامەڵە ژمێریارییە باوەکان و چۆنیەتی کاریگەرییان لەسەر پەرتووکی گشتی (GL).

---

## کارە بازرگانییە ستانداردەکان (ئۆتۆماتیکی)

ئەم تۆمارانە بەزۆری لە ڕێگەی بەڵگەنامە سەرەکییەکانەوە بە شێوەی ئۆتۆماتیکی دروست دەکرێن.

| جۆری مامەڵە | هەژماری قەرزدار (Debit) | هەژماری قەرزدەر (Credit) | کرداری دەستپێکردن |
| :--- | :--- | :--- | :--- |
| **پسووڵەی فرۆشتن** | پارەی کڕیاران (Receivable) | داهات / باجی دەرچوو | پەسەندکردنی پسووڵە |
| **پارەدانی کڕیار** | بانک / کاش | پارەی کڕیاران (Receivable) | تۆمارکردنی پارەدان |
| **پسووڵەی فرۆشیار** | خەرجی / سامان / باجی ناوخۆ | پارەی فرۆشیاران (Payable) | پەسەندکردنی پسووڵە |
| **پارەدانی فرۆشیار** | پارەی فرۆشیاران (Payable) | بانک / کاش | تۆمارکردنی پارەدان |
| **وەرگرتنی کاڵا** | کۆگا (سامان) | هەژماری کاتیی کۆگا (ناوبژیوانی) | پەسەندکردنی وەصڵ |
| **گواستنەوەی کاش** | بانک (وەگر) | بانک (نێرەر) | گواستنەوەی ناوخۆیی |

---

## ڕێکخستنە دەستییە باوەکان

ئەم تۆمارانە بەزۆری بە دەستی لە ڕێگەی مۆدیوڵی **تۆمارەکانی ڕۆژانە** تۆمار دەکرێن.

### 1. داخورانی سامان (Depreciation)
تۆمارکردنی لەدەستدانی بەهای مانگانەی ئامێر یان ئۆتۆمبێل.

| هێڵ | جۆری هەژمار | ناوی هەژمار (نموونە) | قەرزدار | قەرزدەر |
| :--- | :--- | :--- | :--- | :--- |
| 1 | خەرجی | خەرجی داخوران | ✅ | |
| 2 | سامان (پێچەوانە) | داخورانی کۆکراوە | | ✅ |

### 2. خەرجییە شایستەکان (Accrued Expenses)
تۆمارکردنی خەرجییەک کە ئەنجامدراوە بەڵام هێشتا پسووڵەکەی نەهاتووە (نموونە: کرێی کارەبای مانگ).

| هێڵ | جۆری هەژمار | ناوی هەژمار (نموونە) | قەرزدار | قەرزدەر |
| :--- | :--- | :--- | :--- | :--- |
| 1 | خەرجی | خەرجی خزمەتگوزارییەکان | ✅ | |
| 2 | قەرز (قەواڵە) | قەرزە شایستەکان | | ✅ |

### 3. خەرجییە پێشەکییەکان (Prepaid Expenses)
دانی پارەی شتێک پێش وەختە (نموونە: دڵنیایی ساڵانە).

| هێڵ | جۆری هەژمار | ناوی هەژمار (نموونە) | قەرزدار | قەرزدەر |
| :--- | :--- | :--- | :--- | :--- |
| 1 | سامان | دڵنیایی پێشەکی | ✅ | |
| 2 | سامان | بانک / کاش | | ✅ |

### 4. ڕاستکردنەوەی هەڵەی تۆمارکردن
گواستنەوەی مامەڵەیەک لە هەژمارێکی هەڵەوە بۆ هەژمارە ڕاستەکە.

| هێڵ | جۆری هەژمار | ناوی هەژمار (نموونە) | قەرزدار | قەرزدەر |
| :--- | :--- | :--- | :--- | :--- |
| 1 | خەرجی | هەژماری ڕاست (نموونە: پێداویستی ئۆفیس) | ✅ | |
| 2 | خەرجی | هەژماری هەڵە (نموونە: چاککردنەوە) | | ✅ |

---

## یاسا زێڕینەکانی قەرزدار و قەرزدەر

| پۆلێنی هەژمار | زیادبوون (+) | کەمبوون (-) | باڵانسی ئاسایی |
| :--- | :--- | :--- | :--- |
| **سامان (Asset)** | قەرزدار (Debit) | قەرزدەر (Credit) | قەرزدار |
| **قەرز (Liability)** | قەرزدەر (Credit) | قەرزدار (Debit) | قەرزدەر |
| **سەرمایە (Equity)** | قەرزدەر (Credit) | قەرزدار (Debit) | قەرزدەر |
| **داهات (Revenue)** | قەرزدەر (Credit) | قەرزدار (Debit) | قەرزدەر |
| **خەرجی (Expense)** | قەرزدار (Debit) | قەرزدەر (Credit) | قەرزدار |
