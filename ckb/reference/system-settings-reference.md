# سەرچاوەی ڕێکخستنەکانی سیستەم

ئەم سەرچاوەیە وردەکارییەکانی هەڵبژاردەکانی ڕێکخستن لە ڕێکخستنەکانی سیستەم (System Settings) ڕوون دەکاتەوە.

## ڕێکخستنە گشتییەکان

| کێڵگە (Field) | وەسف | نموونە |
| :--- | :--- | :--- |
| **ناوی کۆمپانیا** | ناوی پیشاندراوی کرێگر (Tenant). | Kezi Int. |
| **ناسنامەی باج** | ژمارەی تۆماری باج کە لە بەڵگە یاساییەکان بەکار دێت. | 123-456-789 |
| **ڕەنگی سەرەکی** | ڕەنگی سەرەکی ناسنامە بۆ UI و سەرپەڕی PDFـەکان. | #3b82f6 |
| **لۆگۆ** | پەڕگەی وێنە کە بۆ ناسنامەی بازرگانی بەکار دێت. | `logo.png` |

## بەناوچەییکردن (Localization)

| کێڵگە (Field) | وەسف | نموونە |
| :--- | :--- | :--- |
| **دراوی سەرەکی** | دراوی بنچینە بۆ ڕاپۆرتی دارایی. | USD ($) |
| **فۆرماتی بەروار** | چۆنیەتی نیشاندانی بەروار لە تەواوی سیستەمەکە. | YYYY-MM-DD |
| **کاتی ناوچەیی** | کاتی ناوچەیی (timezone) بۆ کۆمپانیا. | UTC+3 |

## ڕێکخستنەکانی ژمارە-دانان

کۆنترۆڵی چۆنیەتی دروستبوونی ژمارەی بەڵگەکان دەکات.

| ڕێکخستن | نموونەی فۆرمات | وەسف |
| :--- | :--- | :--- |
| **پێشگری فاتوورە** | `INV-2024-` | پێشگر بۆ فاتوورەی فرۆشتن. |
| **پێشگری پسوڵە** | `BILL-2024-` | پێشگر بۆ پسوڵەی دابینکەر. |
| **پڕکردنەوە (Padding)** | `00001` | ژمارەی ڕەقەمەکان لە زنجیرەکە (بۆ نموونە، ٥ ژمارە). |

## ڕێکخستنەکانی PDF

کۆنترۆڵی نەخشەسازی و ناوەڕۆکی PDFـە دروستکراوەکان دەکات.

| هەڵبژاردە | وەسف |
| :--- | :--- |
| **قەبارەی کاغەز** | A4 یان Letter. |
| **نەخشەسازی سەرپەڕ** | لۆگۆ لە چەپ، لۆگۆ لە ڕاست، یان ناوەڕاست. |
| **دەقی پێکەر** | دەقی تایبەت بۆ نیشاندان لە بەشی خوارەوە (بۆ نموونە، وردەکاری بانک). |
| **نیشاندانی ناسنامەی باج** | هەڵکردن/کوژاندنەوە بۆ نیشاندانی ناسنامەی باج لە بەڵگەکان. |
