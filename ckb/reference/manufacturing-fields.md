# سەرچاوە: کێڵگەکان و دۆخەکانی بەرهەمهێنان

ئەم سەرچاوەیە وردەکاری تەکنیکی دەربارەی کێڵگەکان، دۆخەکان، و ڕێکخستنە ژمێریارییەکانی مۆدیوڵی بەرهەمهێنان (MRP) دابین دەکات.

---

## ١. دۆخەکانی فەرمانی بەرهەمهێنان (MO)

لە ئینومی `ManufacturingOrderStatus` دیاری کراوە:

| دۆخ (Status) | مانا | کاریگەری کرداری |
| :--- | :--- | :--- |
| **Draft** | قۆناغی پلاندانان | دەتوانرێت BoM، بڕ و بەروارەکان دەستکاری بکرێت. هیچ کاڵایەک حیجز ناکرێت. |
| **Confirmed** | ئامادە بۆ بەرهەمهێنان | پێکهاتەکان حیجز کراون. دەتوانرێت دەست بە بەرهەمهێنان بکرێت. |
| **In Progress** | بەرهەمهێنان دەستی پێکردووە | پێکهاتەکان بەکاردەهێنرێن. فەرمانەکانی کار چالاک دەبن. |
| **Done** | بەرهەمهێنان تەواو بووە | بەرهەمە تەواوەکان وەرگیراون. پێکهاتەکان بەکارهێنراون. تۆماری ژمێریاری پۆست کراوە. |
| **Cancelled** | هەڵوەشاوە | حیجزەکان لادەبرێن. فەرمانەکانی کار هەڵدەوەشێنرێنەوە. |

---

## ٢. دۆخەکانی فەرمانی کار (Work Order)

لە ئینومی `WorkOrderStatus` دیاری کراوە:

| دۆخ (Status) | مانا | پێویستی |
| :--- | :--- | :--- |
| **Pending** | چاوەڕێی هەنگاوی پێشوو | دەست پێ ناکات تا هەنگاوەکانی پێشوو نەبنە "Done". |
| **Ready** | مەرجەکان جێبەجێ کراون | ناوەندی کار و کەرەستەکان بەردەستن. |
| **In Progress** | کارەکە چالاکە | تایمەرەکە ئیش دەکات بۆ تۆمارکردنی ماوەی ڕاستەقینە. |
| **Done** | هەنگاوەکە تەواو بووە | کردارەکە کۆتایی هاتووە. تێچووەکە تۆمار کراوە. |
| **Cancelled** | کارەکە بازدراوە | کارەکە ئیتر پێویست نییە. |

---

## ٣. جۆرەکانی لیستی کەرەستە (BoM)

| جۆر (Type) | وەسف | باشترین بۆ |
| :--- | :--- | :--- |
| **Normal** | کۆکردنەوەی ستاندارد | ئەو بەرهەمانەی پێویستیان بە هەنگاوی دەستی یان ئامێر هەیە. |
| **Phantom** | گرووپکردنی وەهمی | گرووپکردنی لۆجستی پێکهاتەکان کە پێویستیان بە MO جیا نییە. |
| **Kit** | سێتی فرۆشتن/ناردن | فرۆشتنی چەند بەرهەمێک لەژێر یەک ناوی بەرهەمدا. |

---

## ٤. تۆمارە ژمێریارییەکان (MRP)

سیستەمەکە ئەم تۆمارانە دروست دەکات لە کاتی تەواوبوونی فەرمانی بەرهەمهێنان:

| مامەڵە | قەرزدار (Debit) | خاوەن قەرز (Credit) |
| :--- | :--- | :--- |
| **تەواوبوونی بەرهەمهێنان** | کۆگای بەرهەمی تەواو | کۆگای کەرەستەی خاو |
| **سەرباری ناوەندی کار** | کار لە ژێر جێبەجێکردن (WIP) | سەرباری کارگە |

---

## ٥. پارامێتەرەکانی ناوەندی کار (Work Center)

| پارامێتەر | مانا | کاریگەری |
| :--- | :--- | :--- |
| **کارایی (Efficiency)** | هۆکاری بەکارهێنانی کات | ٢٠٠٪ = ٢ جار خێراتر لە ستاندارد. |
| **توانا (Capacity)** | یەکە لە کاتدا | سنووری پلاندانانی کات دیاری دەکات. |
| **OEE** | کاریگەری گشتی ئامێر | پێوەرە بۆ بەردەستبوون، ئەدا، و جۆرایەتی. |
| **تێچووی کاتژمێر** | نرخی پارە | تێچووی کار/سەرباری بەرهەمەکە هەژمار دەکات. |
