# سەرچاوە: کێڵگەکان و Enumـەکانی کۆگا

ئەم سەرچاوەیە لیستێکە لە کێڵگە سەرەکییەکانی داتا، enumerations، و ڕێکخستنەکان کە لە مۆدیولی کۆگادا بەکاردێن.

---

## ١. کێڵگەکانی ڕێکخستنی بەرهەم

لە کاتی ڕێکخستنی بەرهەمێک بۆ کۆگا، ئەم کێڵگانە ڕەفتارەکان دیاری دەکەن:

| کێڵگە | هەڵبژاردنەکان | وەسف |
| :--- | :--- | :--- |
| **جۆری بەرهەم** | `Storable Product` | شوێنپێی بڕی کۆگا هەڵدەگرێت. |
| | `Consumable` | وا دابنێ هەمیشە بەردەستە (بەدواداچوونی نییە). |
| | `Service` | هیچ بوونێکی فیزیایی نییە. |
| **شێوازی هەڵسەنگاندن** | `FIFO` | یەکەم-هاتوو، یەکەم-ڕۆشتوو. |
| | `LIFO` | کۆتا-هاتوو، یەکەم-ڕۆشتوو. |
| | `AVCO` | تێچووی ناوەندی ڕێکخراو. |
| | `Standard Price` | نرخی جێگیری دەستی. |
| **بەدواداچوون** | `No Tracking` | تەنها بڕی گشتی. |
| | `By Lots` | وەجبەی بەرهەم. |
| | `By Unique Serial` | ناسنامەی تاکەکەسی بۆ هەر یەکەیەک. |

---

## ٢. جۆرەکانی جووڵەی کۆگا (Stock Move Types)

دیاریکراو لە `StockMoveType` Enum:

| جۆر | هێما | وەسف |
| :--- | :--- | :--- |
| **Incoming** | 📥 | دابینکەر ← کۆگا (Inventory Asset زیاد دەکات) |
| **Outgoing** | 📤 | کۆگا ← کڕیار (COGS خەرج دەکات) |
| **Internal** | 🔄 | کۆگای A ← کۆگای B (هیچ گۆڕانکارییەکی ژمێریاری نییە) |
| **Adjustment**| ⚖️ | چاکسازی لە ژمارەی فیزیایی |

---

## ٣. جۆرەکانی شوێنی کۆگا (Stock Location Types)

دیاریکراو لە `StockLocationType` Enum:

| جۆر | سروشت | مەبەست |
| :--- | :--- | :--- |
| **View** | مەجازی | فۆڵدەرێک بۆ کۆکردنەوەی شوێنەکانی تر (ناتوانێت کاڵا هەڵبگرێت). |
| **Internal** | فیزیایی | شوێنێکی فیزیایی کە خاوەندارێتی دەکەیت. |
| **Customer** | مەجازی | نوێنەرایەتی موڵکی کڕیار دەکات. |
| **Vendor** | مەجازی | نوێنەرایەتی موڵکی دابینکەر دەکات. |
| **Inventory Loss** | مەجازی | لایەنی بەرامبەر بۆ کاڵای ونبوو/دۆزراوە. |
| **Production** | مەجازی | لایەنی بەرامبەر بۆ بەکارهێنانی بەرهەمهێنان. |

---

## ٤. تۆمارە ژمێریارییەکان

سیستمەکە ئەم تۆمارانە بە شێوەی خۆکار دروست دەکات:

| مامەڵە | قەرزار (Debit) | قەرزدار (Credit) |
| :--- | :--- | :--- |
| **وەرگرتنی کڕین** | Inventory Asset | Stock Input Account (Interim) |
| **گەیاندنی فرۆشتن** | COGS Account | Inventory Asset |
| **زیادەی کۆگا** | Inventory Asset | Stock Adjustment Gain |
| **زیانی کۆگا** | Stock Adjustment Loss | Inventory Asset |

---

## ٥. ڕاپۆرتەکان

| ڕاپۆرت | مەبەست |
| :--- | :--- |
| **بڕی بەردەست (Stock On Hand)** | بڕی ئێستا بۆ هەر شوێنێک. |
| **هەڵسەنگاندنی کۆگا (Valuation)** | بەهای کۆگای ئێستا ($). |
| **جووڵەی بەرهەم** | مێژووی شوێنپێهەڵگرتن. |
| **تەمەنی کۆگا (Aging)** | دابەشکردنی کۆگا بەپێی تەمەن (٠-٣٠، ٣٠-٦٠...). |
