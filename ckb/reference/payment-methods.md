# ڕێگاکانی پارەدان و بارودۆخەکان

ئەم ڕێنمایییە وردەکاری تەکنیکی بۆ ڕێگاکانی پارەدان و خولی ژیانی پارەدان لە جمیار دەخاتە ڕوو.

## خولی ژیانی بارودۆخی پارەدان

پارەدانەکان بە چەند قۆناغێکدا تێدەپەڕن. تێگەیشتن لەم بارودۆخانە زۆر گرنگە بۆ ڕاپۆرتکردنی دارایی ورد.

| بارودۆخ | وەسف | کاریگەری دارایی |
| :--- | :--- | :--- |
| **Draft** (ڕەشنووس) | پارەدانەکە لە قۆناغی ئامادەکردنیایە و هێشتا هیچ کاریگەرییەکی دارایی نییە. | نییە. |
| **Confirmed** (پەسەندکراو) | پارەدانەکە بە شێوەیەکی ناوخۆیی کۆتایی پێهێنراوە. | کاریگەری لەسەر دەفتەری هاوبەش و ئەژمێرە چاوەڕوانکراوەکان دەبێت. |
| **Reconciled** (یەکخراوەتەوە) | پارەدانەکە لەگەڵ هێڵێکی بەیاننامەی بانک گونجێندراوە. | باڵانسەکە لە ئەژمێری چاوەڕوانکراوەوە بۆ ئەژمێری بانکی دەگوازێتەوە. |
| **Cancelled** (هەڵوەشاوەتەوە) | پارەدانەکە هەڵوەشێنراوەتەوە. | هەر کاریگەرییەکی دارایی پێشوو پێچەوانە دەکاتەوە. |

## جۆرەکانی پارەدان

| جۆر | ئاڕاستە | کارلێکی دەفتەری گشتی |
| :--- | :--- | :--- |
| **Inbound** (بەرەو ناوەوە) | پارەدانی کڕیار | باڵانسی پارە وەرگیراوە چاوەڕوانکراوەکان زیاد دەکات، و باڵانسی ئەژمێری وەرگیراوەکان (A/R) کەم دەکاتەوە. |
| **Outbound** (بەرەو دەرەوە) | پارەدانی دابینکەر | باڵانسی ئەژمێری پارەپێدراوەکان (A/P) کەم دەکاتەوە و باڵانسی پارەدانە چاوەڕوانکراوەکان زیاد دەکات. |

## ڕێگاکانی پارەدان

جمیار پشتیوانی لە چەندین ڕێگا دەکات بۆ پۆلێنکردنی چۆنێتی جوڵەی پارە.

### دەستی (Manual)
ڕێگای ستاندارد بۆ حەواڵەی بانکی، پارەدانی نەختینە، یان هەر پارەدانێک کە لە دەرەوەی سیستمەکە ئەنجام دەدرێت.
- **پرۆسە**: تۆمارکردنی پارەدان لە جمیار -> ئەنجامدانی کارەکە (بۆ نموونە ناردنی چەک) -> یەکخستنەوە کاتێک لە بەیاننامەی بانک دەردەکەوێت.

### چەک (Cheque)
بەکاردێت کاتێک چەکێکی فیزیکی دەردەچێت یان وەردەگیرێت.
- **تایبەتمەندییەکان**: شوێنکەوتنی ژمارەی چەک و کاتی گەیشتن (Maturity Dates).
- **گەیشتن**: پارەدانەکان لە بارودۆخێکی تایبەتدا دەمێننەوە تاوەکو کاتی دیاریکراوی چەکەکە دەگات.

### پارەی بچووک (Petty Cash)
بۆ خەرجییە ناوخۆیییە بچووکەکان بەکاردێت کە لە ڕێگەی سندوقی پارەی بچووکەوە مامەڵەیان لەگەڵ دەکرێت.
- **پرۆسە**: پێویستی بە ڕۆژنامەیەکی تایبەتی پارەی بچووک هەیە.

## سەرچاوەی کێڵگە تەکنیکییەکان (نموونەی پارەدان)

| کێڵگە | جۆر | وەسف |
| :--- | :--- | :--- |
| `amount` | پارە | بڕی پارەدانی بەو دراوەی کە مامەڵەکەی پێ کراوە. |
| `payment_date` | بەروار | ئەو بەروارەی کە پارەدانەکەی تێدا دەستپێکراوە. |
| `journal_id` | ناسنامە | ئەو ڕۆژنامەیەی (بانک یان کاش) کە پارەدانەکەی تێدا تۆمار دەکرێت. |
| `payment_method` | بژاردە | `manual`, `cheque`, `petty_cash`. |
| `reference` | نوسین | ئاماژەی دەرەکی (بۆ نموونە ژمارەی مامەڵەی بانکی). |
