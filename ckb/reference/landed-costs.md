# سەرچاوەی تێچووی گەیشتوو (Landed Cost Reference)

وردەکارییە تەکنیکییەکان لەسەر ڕێگاکانی دابەشکردن، بارودۆخەکانی وۆرکفلۆ، و یەکخستنی ژمێریاری.

---

## ڕێگاکانی دابەشکردن (Allocation Methods)

ڕێگەی دابەشکردن دیاری دەکات چۆن تێچووی گشتی (بۆ نموونە ١,٠٠٠ دۆلار بۆ بارهەڵگرتن) بەسەر چەندین بەرهەمدا دابەش دەکرێت لە باریەکدا.

| ڕێگە | چۆن کاردەکات | باشترین بەکارهێنان بۆ |
| :--- | :--- | :--- |
| **بەپێی بڕ (By Quantity)** | تێچووەکە بە یەکسانی بەسەر هەر پارچەیەکدا دابەش دەکات. | ئەو کاڵایانەی قەبارە و کێشیان لێک نزیکە. |
| **بەپێی نرخ (By Value)** | بەپێی نرخی بەرهەمەکە دابەشی دەکات. | کاڵای گرانبەها بەرامبەر کاڵای هەرزان بۆ پڕکردنەوە. |
| **بەپێی کێش (By Weight)** | بەپێی کێشی فیزیکی بەرهەمەکە دابەشی دەکات. | کرێی بارهەڵگرتن (کە بەپێی کێشە). |
| **بەپێی قەبارە (By Volume)** | بەپێی ئەو شوێنەی دەیگرێت دابەشی دەکات. | بارکردنی کۆنتێنەر (کە شوێن دیاریکەرە). |

---

## بارودۆخەکانی وۆرکفلۆ

| بارودۆخ | وەسف | کاریگەری ژمێریاری |
| :--- | :--- | :--- |
| **ڕەشنووس (Draft)** | تۆمارەکە لە قۆناغی ئامادەکردندایە. | نییە. |
| **پشتڕاستکراوە (Validated)** | تێچووەکان خراونەتە سەر ئینڤێنتۆری. | سەرمایەی ئینڤێنتۆری زیاد دەکات؛ Stock Input زیاد دەکات. |
| **هەڵوەشاوە (Cancelled)** | تۆمارەکە پوچەڵ کراوەتەوە. | تۆمارەکانی پێچەوانەکردنەوە دروست دەبن. |

---

## تۆمارە ژمێریارییەکان

کاتێک تێچوویەکی گەیشتوو **پشتڕاست دەکرێتەوە**، ئەم تۆمارەی خوارەوە دروست دەبێت:

| هەژمار | دەبیت (زیادکردن) | کرێدیت (کەمکردن) |
| :--- | :--- | :--- |
| **سەرمایەی ئینڤێنتۆری** | بڕی $ | — |
| **تێچووی هاتوو - Stock Input (Accrual)** | — | بڕی $ |

*تێبینی: هەژماری Stock Input پاک دەبێتەوە کاتێک پسوڵەی ڕاستەقینەی دابینکەر (Vendor Bill) لە دابینکەری بارهەڵگرتن/دڵنیاییەکەوە تۆمار دەکەیت.*

---

## فەرهەنگۆک (Glossary)

- **تێچووی گەیشتوو (Landed Cost)**: کۆی گشتی نرخی بەرهەمێک دوای ئەوەی دەگاتە بەر دەرگای کڕیار.
- **FOB (Free on Board)**: دیاری دەکات لە چ خاڵێکدا کڕیار دەبێتە بەرپرسیار لە تێچووەکان.
- **DDP (Delivered Duty Paid)**: فرۆشیار هەموو تێچووە گەیشتووەکان لە ئەستۆ دەگرێت.
- **دابەشکردن (Allocation)**: پرۆسە بیرکارییەکەی دابەشکردنی یەک تێچوو بەسەر زۆر کاڵادا.
