# سەرچاوەی گواستنەوەی کۆگا

تایبەتمەندییە تەکنیکییەکان، بارودۆخەکانی وۆرکفلۆ، و چارەسەرکردنی کێشەکان بۆ گواستنەوەکانی نێوان کۆگاکان.

---

## بارودۆخەکانی وۆرکفلۆ (Workflow States)

گواستنەوەکانی نێوان کۆگاکان بەم بارودۆخانەی خوارەوەدا تێپەڕ دەبن:

| بارودۆخ | وەسف | کاریگەری لەسەر کۆگا |
| :--- | :--- | :--- |
| **ڕەشنووس (Draft)** | داواکاری گواستنەوە دروستکراوە، هێشتا دەتوانرێت دەستکاری بکرێت. | هیچ کاریگەرییەکی نییە. |
| **پشتڕاستکراوە (Confirmed)** | کاڵاکە لە کۆگای سەرچاوەدا حیجز (Reserve) کراوە. | حیجزکراو لە سەرچاوە. |
| **نێردراو (Shipped)** | کاڵاکان لە ڕێگادان. | لە ڕێگادا (شوێنی خەیاڵی). |
| **تەواوبوو (Done)** | گواستنەوەکە تەواو بووە و وەرگیراوە. | بەردەست لە شوێنی مەبەست. |

---

## پێویستییەکانی سیستم

### ڕێکخستنی شوێن (Location Configuration)
- **شوێنی سەرچاوە (Source Location)**: دەبێت شوێنێکی ناوخۆیی (Internal) بێت و کاڵای بەردەستی تێدا بێت.
- **شوێنی گواستنەوە (Transit Location)**: شوێنێکی خەیاڵییە کە بۆ چاودێریکردنی کاڵاکان لە کاتی جوڵەکردندا بەکاردێت.
- **شوێنی مەبەست (Destination Location)**: دەبێت شوێنێکی ناوخۆیی بێت.

### پێش مەرجەکان
- بەرهەمی عەمبارکردن (Storable products) کە شوێنپێهەڵگرتنی ئینڤێنتۆرییان بۆ چالاک کراوە.
- بوونی بڕی "بەردەست"ی پێویست لە شوێنی سەرچاوەدا.
- مۆڵەتی بەکارهێنەر بۆ جۆرەکانی کۆکردنەوەی کاڵای "ناوخۆیی" (Internal stock picking).

---

## پێوەرەکانی داشبۆرد و ویتجێتەکان

سیستمەکە چەندین ڕێگە دابین دەکات بۆ چاودێریکردنی گواستنەوەکان:

### ویتجێتی "لە ڕێگادایە" (In-Transit Widget)
- **ژمارە (Count)**: ژمارەی ئەو گواستنەوانەی کە لە بارودۆخی "نێردراو"دان.
- **کۆی نرخ (Total Value)**: نرخی دارایی هەموو ئەو کاڵایانەی کە ئێستا لە ڕێگادان.
- **کۆنی (Aging)**: شیکردنەوەی ئەوەی کە کاڵاکان چەندە لە ڕێگادا ماونەتەوە.

### ڕاپۆرتەکان
- **تێکڕای کاتی گواستنەوە**: ئەو کاتەی کە لە "ناردن"ەوە تا "وەرگرتن" دەخایەنێت.
- **ڕێژەی وردی (Accuracy Rate)**: ڕێژەی سەدی ئەو گواستنەوانەی کە بەبێ جیاوازی بڕ تەواو بوون.

---

## چارەسەرکردنی کێشەکان

| کێشە | هۆکار | چارەسەر |
| :--- | :--- | :--- |
| **دوگمەی "ناردن" (Ship) نابینم** | گواستنەوەکە لە بارودۆخی "پشتڕاستکراوە"دا نییە. | دڵنیابەرەوە لەوەی سەرەتا کلیکت لەسەر "پشتڕاستکردنەوە" کردووە. |
| **دوگمەی "وەرگرتن" (Receive) نابینم** | گواستنەوەکە هێشتا "نەنێردراوە". | دڵنیابەرەوە لەوەی کۆگای نێرەر وەک "نێردراو" نیشانەی کردووە. |
| **ناتوانم دەستکاری بەرهەمەکان بکەم** | گواستنەوەکە پشتڕاستکراوەتەوە یان نێردراوە. | گواستنەوەکە هەڵبوەشێنەرەوە بۆ ئەوەی بگەڕێتەوە بۆ ڕەشنووس (ئەگەر هێشتا نەنێردراوە). |
| **کاڵاکان لە "گواستنەوە"دا دەمێننەوە** | هەنگاوی "وەرگرتن" هەرگیز ئەنجام نەدراوە. | بڕۆ بۆ گواستنەوە نێردراوەکە و کلیک لەسەر وەرگرتن بکە. |
