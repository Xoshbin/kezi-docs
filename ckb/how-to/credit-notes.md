# پسوڵەی گەڕاندنەوە: مامەڵەکردن لەگەڵ گەڕانەوە و ڕاستکردنەوەکان

ئەم ڕێنماییە ڕوونی دەکاتەوە چۆن پسوڵەی گەڕاندنەوە بەکاربهێنیت بۆ کەمکردنەوەی ئەو بڕەی کڕیارەکانت قەرزدارن بۆتۆ. جا گەڕاندنەوەی کاڵا بێت، ڕاستکردنەوەی نرخ بێت، یان هەڵوەشاندنەوەی پسووڵە بێت، پسوڵەی گەڕاندنەوە یارمەتیت دەدات تۆمارەکانی دارایی ڕاست بهێڵیتەوە.

---

## پسوڵەی گەڕاندنەوە چییە؟

**پسوڵەی گەڕاندنەوە** بەڵگەنامەیەکە کە ئەو بڕە کەمدەکاتەوە کە کڕیارێک قەرزدارت بێت. وەک پێچەوانەی پسووڵەی فرۆشتن بیر لێبکەرەوە—لەجیاتی داوای پارە بکەیت، ئەو بڕەی دەبێت بیدەنەوە کەمدەکەیتەوە.

**هۆکارە باوەکان بۆ دەرکردنی پسوڵەی گەڕاندنەوە:**

| هۆکار | نموونە |
|--------|---------|
| **گەڕاندنەوەی کاڵا** | کڕیار ٢ لاپتۆپی خراپ دەگەڕێنێتەوە |
| **ڕاستکردنەوەی نرخ** | پسووڵەی سەرەتایی نرخی هەڵە هەبوو (زۆر بەرز) |
| **داشکاندن دوای پسووڵە** | ڕازیبوون لەسەر ١٠٪ داشکاندن دوای نێردراوی پسووڵە |
| **هەڵوەشاندنەوەی پسووڵە** | کڕیار داواکاری هەڵوەشاند دوای پسووڵەکردن |
| **ڕاستکردنەوەی بڕ** | ٨ دانە گەیەندرا بەڵام ١٠ دانە پسووڵەکرا |

**کاریگەری ژمێریاری**: پسوڵەی گەڕاندنەوەی نێردراو داهات کەمدەکاتەوە و قەرزی وەرگرتن کەمدەکاتەوە.

---

## لەکوێ دەتوانی بیدۆزیتەوە

بڕۆ بۆ: **ژمێریاری ← بەڵگەی ڕاستکردنەوە**

کاتێک بەڵگەی ڕاستکردنەوەی نوێ دروستدەکەیت، **پسوڵەی گەڕاندنەوە** هەڵبژێرە وەک جۆر.

---

## کاروبار و ڕەوتی پسوڵەی گەڕاندنەوە

پسوڵەی گەڕاندنەوەکەت لەم قۆناغانەدا تێدەپەڕێت:

┌─────────┐      ┌───────────┐      ┌───────────┐
│ ڕەشنووس │ ──▶ │ نێردراوە │ ──▶ │ جێبەجێکراو│
└─────────┘      └───────────┘      └───────────┘
    📝               ✅                 💰

### 📝 ڕەشنووس
- دەتوانی هەموو وردەکاریەک بگۆڕیت
- هیچ کاریگەری ژمێریاری نییە هێشتا
- سەلامەتە بۆ سڕینەوە ئەگەر پێویست نەبوو

### ✅ نێردراوە
- داخراوە لە دەستکاری
- تۆماری ڕۆژانە دروستکراوە
- باڵانسی ماوەی کڕیار کەمکراوەتەوە
- ئامادەیە بۆ جێبەجێکردن یان گەڕاندنەوەی پارە

### 💰 جێبەجێکراو
- کرێدیتەکە بەکارهاتووە بۆ کەمکردنەوەی پسووڵەی کراوە
- یان پارەی گەڕاوەتەوە بۆ کڕیار

---

## دروستکردنی پسوڵەی گەڕاندنەوە

### هەنگاوی ١: دەستپێکردن

١. بڕۆ بۆ **ژمێریاری ← بەڵگەی ڕاستکردنەوە**
٢. کلیک بکە لەسەر **دروستکردنی بەڵگەی ڕاستکردنەوە**
٣. **پسوڵەی گەڕاندنەوە** هەڵبژێرە وەک جۆر

### هەنگاوی ٢: پڕکردنەوەی زانیاریەکان

| خانە | چی بنووسیت | نموونە |
|-------|---------------|---------|
| **جۆر** | "پسوڵەی گەڕاندنەوە" هەڵبژێرە | پسوڵەی گەڕاندنەوە |
| **کڕیار** | کێ کرێدیت وەردەگرێت | کۆمپانیای ABC |
| **پارە** | هەمان پارەی پسووڵەی سەرەتا | دۆلار |
| **ڕێکەوت** | ڕێکەوتی پسوڵەی گەڕاندنەوە | ڕێکەوتی ئەمڕۆ |
| **هۆکار** | بۆچی ئەم کرێدیتە دەردەکەیت | "گەڕاندنەوەی ٢ دانەی خراپ" |

### هەنگاوی ٣: زیادکردنی هێڵەکان

کلیک بکە لەسەر **زیادکردنی هێڵ** و بنووسە:

| خانە | وەسف | نموونە |
|-------|-------------|---------|
| **بەرهەم** | کاڵای کرێدیتکراو | لاپتۆپ مودێل X |
| **وەسف** | وردەکاری کرێدیت | "گەڕاندنەوە - شاشەی خراپ" |
| **بڕ** | ژمارەی کاڵاکان | ٢ |
| **نرخی یەکە** | نرخی هەر دانەیەک | ٥٠٠.٠٠ |
| **هەژمار** | هەژماری داهات (خۆکار پڕدەبێتەوە) | داهاتی فرۆشتن |
| **باج** | ئەگەر باج لە پسووڵەی سەرەتادا هەبوو | ١٠٪ باج |

### هەنگاوی ٤: پێداچونەوە و ناردن

پێش کلیککردن لەسەر **ناردن**، دڵنیابەرەوە:

- [ ] کڕیار ڕاستە
- [ ] هێڵەکان لەگەڵ ئەوەی کرێدیتدەکرێت یەکدەگرن
- [ ] بڕ و باجەکان دروستن
- [ ] هۆکار بۆ پشکنین بەڵگەکراوە

کلیک بکە لەسەر **ناردن** بۆ کۆتاییپێهێنان.

---

## چی ڕوودەدات لەپشت پەردە

کاتێک پسوڵەی گەڕاندنەوەیەک دەنێریت بۆ گەڕاندنەوەی $١,٠٠٠ (لەگەڵ ١٠٪ باج)، سیستەم ئەمە دروستدەکات:

```
┌─────────────────────────────────────────────────────────────┐
│  قەرزدار. داهاتی فرۆشتن                $1,000.00            │
│  قەرزدار. باجی فرۆشتن                    $100.00            │
│      قەرزدەر. قەرزی وەرگرتن                     $1,100.00   │
└─────────────────────────────────────────────────────────────┘
```

**بە کوردی ئاسان**: ئێمە فرۆشتنەکەی سەرەتا پێچەوانەدەکەینەوە—داهاتی تۆمارکراومان کەمدەکەینەوە و ئەوەی کڕیار قەرزدارمانە کەمدەکەینەوە.

---

## باڵەخانەی باو

### باڵەخانەی ١: کڕیار کاڵای خراپ دەگەڕێنێتەوە

**دۆخەکە**: کڕیارێک ١٠ لاپتۆپی کڕی بە $٥٠٠ هەر یەکەیان. دوو دانەیان خراپ بوون و گەڕێندرانەوە.

**چی بکەیت**:

١. بەڵگەی ڕاستکردنەوەی نوێ دروستبکە ← جۆر: **پسوڵەی گەڕاندنەوە**
٢. کڕیارەکە هەڵبژێرە
٣. پەیوەندیکردن بە پسووڵەی سەرەتا (ئارەزوومەندانە)
٤. هێڵ زیادبکە:
   - بەرهەم: لاپتۆپ
   - بڕ: ٢
   - نرخ: $٥٠٠
   - باج: هەمان پسووڵەی سەرەتا
٥. هۆکار: "گەڕاندنەوەی ٢ دانەی خراپ"
٦. پسوڵەی گەڕاندنەوەکە **بنێرە**

**ئەنجام**: 
- باڵانسی کڕیار بە $١,١٠٠ کەمکراوەتەوە (لەگەڵ باج)
- داهاتی فرۆشتن بە $١,٠٠٠ کەمبووەتەوە
- قەرزی باج بە $١٠٠ کەمبووەتەوە

### باڵەخانەی ٢: ڕاستکردنەوەی نرخ

**دۆخەکە**: پسووڵەت کرد بۆ کڕیارێک بە $٢٠٠/یەکە بەڵام نرخی ڕازیبووی سەرەکە $١٨٠/یەکە بوو بۆ ٥٠ دانە.

**چی بکەیت**:

١. پسوڵەی گەڕاندنەوە دروستبکە
٢. جیاوازییەکە حیساب بکە: ($٢٠٠ - $١٨٠) × ٥٠ = **$١,٠٠٠**
٣. هێڵ زیادبکە بە جیاوازی نرخ
٤. هۆکار: "ڕاستکردنەوەی نرخ بەپێی ڕازیبوون"
٥. **بنێرە**

---

## ڕێنمایی باش

### ✍️ هەمووشتێک بەڵگەبکە
هەمیشە **هۆکار**ی ڕوون لەخۆبگرە کە ڕوونی دەکاتەوە بۆچی کرێدیتەکە دەرکراوە.

### 🔗 پەیوەندی بکە بە بەڵگەنامەی سەرەتاوە
کاتێک دەتوانی، پسوڵەی گەڕاندنەوەکە پەیوەندیبکە بە پسووڵەی سەرەتاوە. ئەمە ڕەگەزنامەی پشکنین دروستدەکات.

### 📋 ژمارەی بەڵگە بەکاربهێنە
ژمارەی گەڕاندنەوە، پەیوەندی گلەییەکانی کڕیار، یان ژمارەی شوێنپێچی تردا لە وەسفەکەدا.

### 📊 پێداچوونەوە پێش ناردن
کاتێک نێردرا، پسووڵەکانی گەڕاندنەوە ناتوانرێت دەستکاری بکرێن. هەموو بڕەکان دووبارە بپشکنە پێش پەسەندکردن.

---

## چارەسەرکردنی کێشەکان

### هەڵەی "ناتوانرێت بنێردرێت"

**بپشکنە**:
- ئایا هەموو خانە پێویستەکان پڕکراونەوە؟
- ئایا ڕێکەوتەکە لەناو ماوەی ژمێریاری کراوەدایە؟
- ئایا پارەکە دەستەبەر و چالاکە؟

### پسوڵەی گەڕاندنەوە باڵانس کەمناکاتەوە

**بپشکنە**:
- ئایا پسوڵەی گەڕاندنەوە نێردراوە (نەک تەنها وەک ڕەشنووس پاشەکەوتکراوە)؟
- ئایا پەیوەندیکراوە بە کڕیاری دروستەوە؟
