# تۆمارکردنی تۆمارە دەستییەکانی ڕۆژانە

هەرچەندە زۆربەی تۆمارە داراییەکان بە شێوەی ئۆتۆماتیکی دروست دەکرێن، بەڵام ڕەنگە پێویستت بە تۆمارکردنی تۆمارە دەستییەکان بێت بۆ ڕێکخستنەوە، ڕاستکردنەوە، یان مامەڵە نائاساییەکان (وەک داخوران).

---

## کەی تۆماری دەستی بەکاردەهێنیت

ئەم ڕێبەرە بەکاربهێنە کاتێک پێویستت بەم کارانە بێت:
1.  **ڕاستکردنەوەی هەڵەکان**: گۆڕینی پۆلێنی خەرجییەک بۆ هەژمارێکی جیاواز.
2.  **تۆمارکردنی داخوران (Depreciation)**: تۆمارکردنی لەدەستدانی بەهای مانگانەی داراییەکان (وەک ئامێر یان ئۆتۆمبێل).
3.  **شایستەکان و پێشەکییەکان (Accruals & Prepayments)**: ڕێکخستنی ئەو داهات یان خەرجییانەی کە سەر بە ماوەی جیاوازن.
4.  **باڵانسەکانی کردنەوە (Opening Balances)**: تۆمارکردنی باڵانسی سەرەتایی هەژمارەکانت لە کاتی دەستپێکردن بە سیستەمەکە.

---

## دروستکردنی تۆمارێکی دەستی

### 1. چوون بۆ مۆدیوڵەکە
بڕۆ بۆ **ژمێریاری ← ژمێریاری ← تۆمارەکانی ڕۆژانە** و کلیک لە **دروستکردن (Create)** بکە.

### 2. داخلکردنی زانیارییە سەرەکییەکان
زانیارییە بنەڕەتییەکانی تۆمارەکە پڕ بکەرەوە:

| بوار | وەسف | نموونە |
| :--- | :--- | :--- |
| **ڕۆژنامە (Journal)** | ئەو "پەرتووکە" تایبەتەی بۆ ئەم تۆمارە بەکاردێت. | * Miscellaneous Operations* |
| **سەرچاوە (Reference)** | ناسنامەیەکی تایبەت یان وەسفێک بۆ ڕێکخستنەوەکە. | *ADJ-2024-001* |
| **ڕێکەوت** | ڕێکەوتی کاریگەری مامەڵەکە. | *2024-01-31* |

### 3. زیادکردنی بڕگەکانی ڕۆژانە (یاسای دوو-تۆمار)
هێڵەکان بۆ تۆمارەکە زیاد بکە. هەر تۆمارێک دەبێت لانی کەم دوو هێڵی هەبێت، و **کۆی قەرزدار دەبێت یەکسان بێت بە کۆی قەرزدەر**.

1.  **هەڵبژاردنی هەژمار**: ئەو هەژمارەی پەرتووکی گشتی کە کاریگەری لەسەرە هەڵیبژێرە.
2.  **هاوبەش (بژاردە)**: فرۆشیار یان کڕیارێک هەڵبژێرە ئەگەر هێڵەکە پەیوەندی پێیانەوە هەبێت.
3.  **ناونیشان/تێبینی (Label)**: وەسفێکی ڕوون بۆ ئەم هێڵە بنووسە.
4.  **قەرزدار / قەرزدەر (Debit / Credit)**: بڕی پارەکە داخل بکە.

> [!TIP]
> بوارەی **ناونیشان (Label)** بەکاربهێنە بۆ ڕوونکردنەوەی ئەوەی *بۆچی* ئەم هێڵە تۆمار دەکرێت. ئەمە لە کاتی وردبینیدا زۆر یارمەتیدەر دەبێت.

### 4. پشتڕاستکردنەوە و ناردن
1.  **پشکنینی هاوسەنگی**: دڵنیابەرەوە کە "جیاوازی" (Difference) لە خوارەوەی خشتەکە **0.00** بێت.
2.  **پاشەکەوتکردن وەک ڕەشنووس**: کلیک لە **دروستکردن** بکە. ئێستا تۆمارەکە لە بارودۆخی **ڕەشنووس (Draft)** دایە و هێشتا کاریگەری لەسەر داراییەکانت نییە.
3.  **ناردن (Post)**: دوای پێداچوونەوە، کلیک لە **Post** بکە. ئێستا تۆمارەکە کۆتایی پێهاتووە و قوفڵ بووە.

---

## پێچەوانەکردنەوەی تۆمارێکی هەڵە

ئەگەر تۆمارێکت **Post** کردووە و دواتر هەڵەیەکت دۆزییەوە، ناتوانیت دەستکاری بکەیت. دەبێت پێچەوانەی بکەیتەوە:

1.  تۆمارە نێردراوەکە (Posted) بکەوە.
2.  کلیک لە دوگمەی **پێچەوانەکردنەوە (Reverse)** بکە لە سەرەوە.
3.  ڕێکەوتی پێچەوانەکردنەوە دیاری بکە (زۆربەی کات هەمان ڕێکەوتی سەرەکی یان ئەمڕۆیە).
4.  سیستەمەکە تۆمارێکی نوێ دروست دەکات کە بەهاکانی قەرزدار/قەرزدەر گۆڕدراون بۆ پووچەڵکردنەوەی تۆمارە سەرەکییەکە.
