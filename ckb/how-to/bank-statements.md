# ڕاپۆرتەکانی هەژماری بانک: بنەمای بەڵگەکردنی گەرانی پارە و هاوسەنگی بانکی

ئەم ڕێبەرییە گشتگیرە ڕێنوێنیت دەکات چۆن ڕاپۆرتی هەژماری بانک دروستبکەیت لە سیستەمی ژمێریاریدا لەگەڵ تۆمارکردنی هێڵەکانی مامەڵە بانکییەکان، مامەڵەکردن بە پارە جیاوزەکان، چۆن یەکدەگرێتەوە لەگەڵ هاوتاکردنی [هاوتاکردنی بانکی](bank-reconciliation.ckb.md). ئەم ڕێبەرییە نووسراوە بۆ هەموو بەکارهێنەرێک — ژمێریار بێت یان بەکارهێنەری ئاسایی — لەم ڕێبەرییەدا هەوڵدراوە بە نمونەی کردەیی هەنگاوەکان باس بکرێت کە پەیڕەوی لە باشترین شێوازەکانی ژمێریاری دوو-تۆماری دەکات.

**گرنگ**: هەموو مامەڵە بانکییەکان پێویستییان بە ڕاپۆرتی هەژماری بانکی دروست هەیە بۆ درێژەدان بە پرۆسەی هاوتاکردن..

---

## ڕاپۆرتەکانی هەژماری بانک چین؟

هەموو ئەو مامەڵانە دەگرێتەوە کە گواستنەوەی پارە لە لایەنێکەوە بۆ لایەنێکی تر ڕویانداوە لە ڕێی بانکەوە لە ماوەیەکی دیاریکراودا. بە واتایەکی ئاسانتر ئەو مامەڵانە دەگرێتەوە کە بانک نێوەندگیر بووە لە نێوان فرۆشیار و کڕیاردا بە نموونەیەکی ئاسان کاتێک کاڵایەک دەکڕیت لە ڕێی کرێدت کارتەوە بانک نێوەندگیرەو پێی دەوترێت مامەڵەی بانکی.

-   **تۆماری مامەڵەکان**: تۆمارکردنی مامەڵە بانکییە هاتووەکان، رۆشتووەکان، ڕاکێشراوەکان، تێچووی مامەڵەکان
-   **دڵنیابوونەوە لە هاوسەنگی**: چاودێریکردنی یەکەی سەرەتاو کۆتایی هەر وادەیەک
-   **هاوتاکردنی بانکی**: دڵنیابوونەوە لە هاوتاکردنی مامەڵە بانکییەکان لەگەڵ تۆماری پارەدانەکان
-   **چاودێریکردنی ئاڕاستەی پارە(کاش)**: چالاککردنی چاودێری وورد و دروستی جوڵەی کاش

**بنەمای ژمێریاری**: ڕاپۆرتی هەژماری بانکی وەک سەرچاوەیەکی دڵنیابونەوەی سەربەخۆ خزمەتی مامەڵە داراییەکان دەکات، یارمەتیدەر دەبێت لە دڵنیابوونەوە لە هاوتاکردنی یان دۆزینەوەی جیاوازییەکانی تۆماری پارەدانەکان بەراورد بە مامەڵە بانکییەکان لە ڕێی هاوتاکردنی بانکییەوە.

---

## پێداویستیەکانی سیستەم

### ڕێکخستنی پەرتووکی ڕۆژانە
-   **پەرتووکی ڕۆژانەی بانک**: پێویستە بە جۆری "بانک" ڕێکخرابێت
-   **ڕێکخستنی پارە**: پارەی پەرتووکی ڕۆژانە دەبێت لەگەڵ پارەی ڕاپۆرتەکە یەکبگرێتەوە
-   **ڕێکخستنی هەژمار**: هەژماری بانک دەبێت بە دروستی بە پەرتووکی ڕۆژانەوە بەسترابێتەوە

### پێشمەرجەکان
1.   **دانانی پەرتووکی ڕۆژانەی بانک**: لانیکەم یەک پەرتووکی ڕۆژانەی بانک دەبێت ڕێکخرابێت (سیستمی ژمێریار خۆشبەختانە هەژمارە بنەڕەتە سەرەتاییەکانی تێدایە)
2.  **مۆڵەتەکانی بەکارهێنەر**: دەستگەیشتنی گونجاو بە تایبەتمەندییەکانی بانک و بەڕێوەبردنی کاش
3.  **ڕێکخستنی کۆمپانیا**: تایبەتمەندییەکانی بانک لە ڕێکخستنەکانی کۆمپانیادا چالاک کرابێت

---

## چۆن لە ڕووکاری بەکارهێنەردا بیدۆزینەوە

بڕۆ بۆ **ژمێریاری → بانک و کاش → ڕاپۆرتەکانی هەژماری بانک**

ڕاپۆرتەکانی هەژماری بانک لەم بەشانەشدا دەردەکەون:
-   **کرداری دروستکردن**: دوگمەی "دروستکردن" بۆ تۆماری ڕاپۆرتی نوێ
-   **لیستی ڕاپۆرت**: پێداچوونەوەی هەموو ڕاپۆرتەکان لەگەڵ بارودۆخی هاوتاکردن
-   **تۆمارەکانی پارەدان**: زانیاری پەیوەندیدارەکان بە ڕاپۆرتی هەژماری بانک

**تێبینی**: دوگمەی یارمەتی/بەڵگەنامە لە ڕووکاری بەکارهێنەردا ئەم ڕێبەرییە دەکاتەوە.

---

## دروستکردنی ڕاپۆرتەکانی هەژماری بانک

بڕۆ بۆ **ژمێریاری → بانک و کاش → ڕاپۆرتەکانی هەژماری بانک → دروستکردن**

### هەنگاوی ۱: زانیاری بەشی سەرەوەی ڕاپۆرت

**پارە**: پارەی ڕاپۆرتەکە هەڵبژێرە (بە شێوەی خۆکارانە پارەی بنەڕەتی کۆمپانیا هەڵبژێردراوە)
**پەرتووکی ڕۆژانەی بانک**: پەرتووکی ڕۆژانەی بانک هەڵبژێرە (پێویستە جۆری "بانک" بێت)
**ژمارەی بەڵگەنامە**: ژمارەی ڕاپۆرت یان ناسێنەرێک دابنێ (نموونە، "STMT-2024-001")
**بەروار**: بەرواری ڕاپۆرتەکە دابنێ
**یەکەی دەستپێکردن**: یەکەی دەستپێک لە ڕاپۆرتی پێشووترەوە دابنێ
**یەکەی کۆتایی**: یەکەی کۆتایی لە ڕاپۆرتی ئێستاوە دابنێ

### هەنگاوی ۲: تۆمارکردنی مامەڵە بانکییەکان

**بەروار**: بەرواری مامەڵە کە سەرچاوەکەی ڕاپۆرتی هەژماری بانکە
**شرۆڤە**: شرۆڤەی بانک بۆ مامەڵەکە
**هاوبەش**: بەستنەوە بە کڕیار/فرۆشیار ئەگەر گونجاو بێت (داواکراو نییە)
**بڕ**: بڕی مامەڵە
-   بڕی ئەرێنی بۆ وەرگرتن، گواستنەوەی ناوخۆ، سوودی وەرگیراو
-   بڕی نەرێنی بۆ ڕاکێشان، چێک، کرێ، گواستنەوەی دەرەوە

**هەڵبژاردنی پارەی بیانی** (داواکراو نییە بەڵام بۆ ئەو مامەڵانەی بە پارەی جیاواز کراون پێویستە دیاریبکرێت):
-   **پارەی بیانی**: پارەی ڕەسەنی مامەڵەکە هەڵبژێرە
-   **بڕ لە پارەی بیانیدا**: بڕی ڕەسەنی مامەڵە  پێش گۆڕین تۆماربکە

### هەنگاوی ۳: پشتڕاستکردنەوە و پاشەکەوتکردن

**پشتڕاستکردنەوەی هاوسەنگی**: سیستەم خۆکارانە یەکەی کۆتایی دەژمێرێت
**پێداچوونەوەی هێڵەکان**: دڵنیابەرەوە هەموو مامەڵەکان بە دروستی تۆمارکراون
**پاشەکەوتکردنی ڕاپۆرت**: ڕاپۆرتەکە دروست بکە و بۆ هاوتاکردنی بانکی ئامادەی بکە

---

## ڕاپۆرتەکانی هەژماری بانکی فرە-پارە

### پشتگیریکردنی مامەڵەی پارەی بیانی

**پارەی ڕاپۆرت**: پێویستە بڕی هەموو مامەڵەکان بە پارەی بنچینەیی ڕاپۆرتە بانکییەکە تۆماربکرێن
**پارەی ڕەسەن**: بەدواداچوونی بۆ پارەی ڕەسەنی مامەڵەکە لەگەڵ بڕەکەی
**نرخی ئاڵوگۆڕ**: نرخی گۆڕین لە کاتی تۆمارکردنی مامەڵەدا تۆماردەکرێت
**پیشاندان**: هەردوو بڕی ڕەسەن و گۆڕدراو لە تۆمارەکاندا دیاردەبن

### هەژمارەکانی بانکی فرە-پارە

**هاوتاکردنی پارە**: هەموو ڕاپۆرتێک دەبێت لەگەڵ پارەی پەرتووکی ڕۆژانە یەکبگرێتەوە
**نرخی ئاڵوگۆڕ**: لە کاتی تۆمارکردنی مامەڵەکاندا تۆمار دەکرێن
**هاوتاکردنی بانکی**: جیاوازییەکانی پارە لە کاتی هاوتاکردندا خۆکارانە دەردەکەون

**نموونە**: ڕاپۆرتی هەژماری بانکی دۆلاری ئەمریکی لەگەڵ مامەڵەی یۆرۆ:
-   پارەی ڕاپۆرت: دۆلاری ئەمریکی
-   مامەڵە: -$95.50 (لە -€85.00 گۆڕدراوە بە نرخی 1.1235)
-   بڕی ڕەسەن: €85.00 یۆرۆ
-   بڕی گۆڕدراو: $95.50 دۆلاری ئەمریکی

## پشتڕاستکردنەوەی ڕاپۆرت

### پڕۆسەی پشتڕاستکردنەوەی هاوسەنگی

**یەکەی کۆتایی ژمێردراو** = یەکەی دەستپێک + کۆی گشتی هەموو مامەڵەکان
**دەبێت یەکسان بێت بە**: یەکەی کۆتایی بانک لە ڕاپۆرتەکەدا

### کێشە باوەکانی پشتڕاستکردنەوە

**ناهاوسەنگی**: یەکەی ژمێردراو یەکسان نییە بە یەکەی کۆتایی ڕاپۆرت
-   پێداچوونەوە بە هەموو بڕە پارەکاندا و ووردبونەو لە هێماکانی مامەڵە
-   ووردبینی کردن  لە تۆمارە ونبووەکان یان تۆمارە دووبارەکاندا
-   پشتڕاستکردنەوەی وردی یەکەی دەستپێک

**مامەڵە ونبووەکان**: ڕاپۆرتەکە ناتەواو دەردەکەوێت
-   ووردبینی لە تۆمارەکاری وەرگرتن یان ڕاکێشاندا بکەو تۆمارنەکراوەکان زیاد بکە
-   ووردبینی لە کرێ یان خەرجییەکان لە ڕاپۆرتەکەدا
-   پشتڕاستکردنەوەی هەموو بەروارەکانی مامەڵە

**تۆمارە دووبارەکان**: هەمان مامەڵە دوو جار تۆمارکراوە
-   تۆمارە یان هێڵە دووبارەکان لە ڕاپۆرتەکەدا ڕەشبکەرەوە
-   پشکنین بۆ بڕە هاوشێوەکان بکە کە لە هەمان بەرواردا تۆمارکراون

## سیناریۆ باوەکان

### سیناریۆی ۱: دروستکردنی ڕاپۆرتی مانگانە
دروستکردنی ڕاپۆرتی تەواوی بانکی مانگانە:

**هەنگاوەکان**:
1.  بڕۆ بۆ ڕاپۆرتەکانی هەژماری بانک → دروستکردن
2.  پارەی دیناری عێراقی و پەرتووکی ڕۆژانەی بانکی سەرەکی هەڵبژێرە
3.  ژمارەی بەڵگەنامەی ڕاپۆرت "STMT-2024-03" و بەروار دابنێ
4.  یەکەی دەستپێک دابنێ: 5,000,000 دیناری عێراقی
5.  25 هێڵی مامەڵە زیاد بکە (وەرگرتن و ڕاکێشان)
6.  دڵنیابەرەوە یەکەی کۆتایی ژمێردراو لەگەڵ بانکی یەکدەگرێتەوە: 4,750,000 دیناری عێراقی
7.  ڕاپۆرتەکە پاشەکەوت بکە و بۆ هاوتای ئامادەی بکە

**ئەنجام**:
-   ڕاپۆرتی تەواوی بانک بۆ هاوتاکردنی بانکی ئامادەیە
-   هەموو مامەڵەکان بە شرۆڤەی گونجاو و بەڵگە تۆمارکراون
-   دڵنیابونەوە کە  هاوسەنگی تێپەڕ دەبێت

### سیناریۆی ۲: تۆمارکردنی مامەڵەی فرە-پارە
تۆمارکردنی مامەڵەیەکی پارەی بیانی لە ڕاپۆرتێکی دیناری عێراقی:

**نموونە**:
-   پارەی ڕاپۆرتی بانک: دیناری عێراقی
-   مامەڵە: پارەدانی گواستنەوەی بانکی دۆلاری ئەمریکی
-   بڕی ڕەسەن: -$500 دۆلاری ئەمریکی
-   نرخی ئاڵوگۆڕ: 1 دۆلاری ئەمریکی = 1,310 دیناری عێراقی
-   بڕی گۆڕدراو: -655,000 دیناری عێراقی

**هەنگاوەکان**:
1.  هێڵێکی مامەڵە زیاد بکە بە بڕی: -655,000 دیناری عێراقی
2.  دۆلاری ئەمریکی وەک پارەی بیانی هەڵبژێرە
3.  -500 وەک بڕی پارەی بیانی دابنێ
4.  سیستەم هەردوو بڕەکە بۆ بەدواداچوون دەپارێزێت
5.  پاشەکەوتکردن وەک بەڵگەنامەی دوو-پارەی

**ئەنجام**:
-   مامەڵە بە پارەی ڕەسەنی ڕاپۆرتەکە (دیناری عێراقی) تۆمارکرا
-   جۆری پارە و بڕی پارەی ڕەسەن پارێزراون
-   بەڵگەنامەی نرخی ئاڵوگۆڕ بەردەستە

### سیناریۆی ۳: ڕاستکردنەوەی هەڵە
ڕاستکردنەوەی بڕی مامەڵەیەکی هەڵە پێش هاوتاکردنی بانکی:

**هەنگاوەکان**:
1.  ڕاپۆرتەکە لە شێوازی دەستکاریکردندا بکەوە یان بیکەرەوە بۆ دەستکاریکردن
2.  هێڵی مامەڵە هەڵەکە بدۆزەرەوە (بڕی پارە هەڵە)
3.  بڕی پارە هەڵەکە بگۆڕە بە بڕی دروست
4.  دڵنیابەرەوە ژماردنی یەکەی کۆتایی خۆکارانە نوێ دەبێتەوە
5.  گۆڕانکارییەکان پاشەکەوت بکە
6.  ئەگەر پێشتر هاوتا کراوە، لەبری ئەوە پێویستە تۆماری ڕاستکەرەوە دروست بکەیت

**ئەنجام**:
-   دروستی ڕاپۆرت پارێزراوە
-   پشتڕاستکردنەوەی هاوسەنگی دروست دەمێنێتەوە
-   ڕێڕەوی پشکنین پارێزراوە

## باشترین شێوازەکان

### وردی لە تۆمارکردنی داتا
-   **دووجار پشکنینی بڕەکان**: پشتڕاستکردنەوەی هەموو بڕە پارەکان و نیشانەکانی مامەڵەکان
-   **شرۆڤەی دروست**: شرۆڤەی مانادار لە تۆماربکە دەربارەی مامەڵە بانکییەکان
-   **بەستنەوەی هاوبەش**: بەستنەوەی مامەڵەکان بە کڕیار/فرۆشیار کاتێک گونجاو بێت
-   **پارەی بیانی**: تۆمارکردنی بڕە ڕەسەنەکان بۆ مامەڵەکانی فرە-پارە

### بەڕێوەبردنی ڕەوتی کار
-   **تۆمارکردنی بەردەوام**: پێویستە ڕاپۆرتەکانی بانک هەرکات بە دەستمان گەیشت دەستبەجێ تۆماربکرێن
-   **پێداچوونەوەی سیستماتیک**: پشتڕاستکردنەوەی هاوسەنگی ژمێردراو پێش پاشەکەوتکردن
-   **هاوتاکردنی دەستبەجێ**: پڕۆسەی هاوتاکردن پێویستە دەستبەجێ دوای دروستکردن دەستپێبکرێت
-   **ڕێگرتن لە هەڵە**: پێداچوونەوە بە هەموو تۆمارەکان پێش کۆتاییپێهێنان

### ستانداردەکانی بەڵگەنامەکردن
-   **ژمارەی بەڵگەی ڕوون**: بەکارهێنانی ژمارەی بەڵگەی هاوشێوە (نموونە، STMT-YYYY-MM)
-   **وردەکاری مامەڵە**: لەخۆگرتنی شرۆڤەی بانک لەگەڵ سەرنج و تێبینی خۆت
-   **وردی بەروار**: دڵنیابەرەوە بەروارەکانی مامەڵە بە تەواوی لەگەڵ ڕاپۆرتی بانک یەکدەگرنەوە
-   **ڕوونی پارە**: تۆمارکدنی مامەڵەکانی پارەی بیانی بە تەواوی

### پشکنین و پابەندبوون
-   **تۆمارە کۆتاهاتووەکان**: پاراستنی هەموو زانیاریی و ڕاپۆرتەکان بۆ کاتی پشکنین و پێداچونەوە
-   **چاودێریکردنی گۆڕانکاری**: سیستەم مێژووی گۆڕانکارییەکانی ڕاپۆرت دەپارێزێت
-   **کۆنترۆڵی دەستگەیشتن**: سنووردارکردنی دروستکردنی ڕاپۆرت بۆ کارمەندانی ژمێریاری مۆڵەتپێدراو
-   **پڕۆسەی پێداچوونەوە**: جێبەجێکردنی ڕەوتی کارگێڕی پەسەندکردن بۆ مامەڵە گەورەکان

---

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**پرسیار: ناتوانرێت پەرتووکی ڕۆژانەی بانک لە لیستەکەدا هەڵبژێردرێت**
وەڵام: ڕێکخستنی پەرتووکی ڕۆژانە و دەستگەیشتن بپشکنە:
-   دڵنیابەرەوە جۆری پەرتووکی ڕۆژانە وەک "بانک" ئامادەکراوە
-   دڵنیابەرەوە پەرتووکی ڕۆژانە سەر بە کۆمپانیا ئێستایە
-   مۆڵەتەکانی بەکارهێنەر بۆ دەستگەیشتن بە پەرتووکی ڕۆژانە بپشکنە
-   دڵنیابەرەوە کە  کە پەرتووکی ڕۆژانە چالاکەوە و ئەرشیڤ نەکراوە

**پرسیار: ژماردنی هاوسەنگی لەگەڵ ڕاپۆرتی هەژماری بانک یەکناگرێتەوە**
وەڵام: پێداچونەو بکە بە دروستی تۆمارەکاندا:
-   هەموو بڕە پارەکان و نیشانەکانی مامەڵە (ئەرێنی/نەرێنی) بپشکنە
-   بگەڕێ بە دوای تۆمارە ونبووەکان یان دووبارەبووەکان
-   دڵنیابەرەوە یەکەی دەستپێک لە هاوتایە لەگەڵ یەەی کۆتایی ڕاپۆرتی ڕابردوو
-   دڵنیابەرەوە تۆماری بڕی ئاڵوگۆڕی پارە بیانییەکان دروستن

**پرسیار: ناتوانرێت ڕاپۆرت دەستکاری بکرێت دوای دروستکردن**
وەڵام: بارودۆخی هاوتای بانکی و مۆڵەتەکان بپشکنە:
-   دڵنیابەرەوە هێڵەکانی ڕاپۆرتەکە هێشتا هاوتانەکراون
-   پشتڕاستکردنەوەی مۆڵەتەکانی بەکارهێنەر بۆ دەستکاریکردنی ڕاپۆرتەکانی بانک
-   بیر لە دروستکردنی ڕاپۆرتی ڕاستکەرەوە بکەرەوە لەبری دەستکاریکردن
-   ووردبونەوە لە ئەگەری ڕاپۆرتەکە بەهۆی داخستنی ماوەی ژمێریارییەوە قوفڵ کرابێت

**پرسیار: بڕەکانی پارەی بیانی بە دروستی ناژمێردرێن**
وەڵام: دنیابەرەوە لە ئامادەکردنی پارەی پێویست و تۆمارکردنی نرخەکانی ئاڵوگۆڕی پارە:
-   دڵنیابەرەوە هەردوو پارە چالاک و ئامادەن
-   دلیابەرەوە کە نرخی ئاڵوگۆڕی پارە لە ڕۆژی مامەڵەکەدا تۆمارکراوە
-   دڵنیابەرەوە بڕی ڕەسەن بە تەواوی لەگەڵ ڕاپۆرتی بانک یەکدەگرێتەوە
-   پێداچوونەوە بە ژماردنی گۆڕینی پارە

### پەیامەکانی هەڵە

**"یەکەی دەستپێک ناتوانرێت دەستکاری بکرێت"**
-   یەکەی دەستپێک بە شێوەی ئاسایی لە ڕاپۆرتی پێشووترەوە دەمێنێتەوە
-   دڵنیابەرەوە کە ڕاپۆرتی بەردەست کە هەڵەی تێدایە یەکەم ڕاپۆرتە یان نا
-   دڵنیابەرەوە ڕێکخستنی هەژمارەکە بە دروستی ئەنجامدراوە

**"جۆری پەرتووکی ڕۆژانە دەبێت بانک بێت"**
-   پەرتووکی ڕۆژانەی هەڵبژێردراو وەک جۆری بانک ڕێکنەخراوە
-   پەرتووکی ڕۆژانەی جیاواز هەڵبژێرە یان ڕێکخستنی پەرتووکی ڕۆژانە نوێ بکەرەوە
-   پەیوەندی بە بەڕێوەبەری سیستەمەوە بکە بۆ دانانی پەرتووکی ڕۆژانەی بانکی گونجاو

## پرسیارە دووبارەبووەکان

**پرسیار: چەند جار دەبێت ڕاپۆرتەکانی هەژماری بانک دروست بکەم؟**
وەڵام: ڕاپۆرتەکان دروست بکە هەرکاتێک لە بانکەوە وەرتگرتن، بە شێوەی ئاسایی مانگانە. ڕاپۆرتەکانی زۆرتر (هەفتانە یان ڕۆژانە) دەتوانرێن بۆ هەژمارە چالاکەکان دروست بکرێن کە پێویستیان بە چاودێری نزیکتر هەیە.

**پرسیار: چی ڕوودەدات ئەگەر دوای هاوتای هەڵەیەکم کرد؟**
وەڵام: هەرکاتێک هێڵەکانی ڕاپۆرت هاوتا کران، خۆت بەدوور بگرە لە دەستکاریکردنی ڕاپۆرتە ڕەسەنەکە. لەبری ئەوە، ڕاپۆرتێکی نوێ دروست بکە بە تۆمارەکانی ڕاستکەرەوە بۆ پاراستنی یەکپارچەیی پشکنین.

**پرسیار: دەتوانم ڕاپۆرتەکانی بانکی خۆکارانە هاوردە بکەم؟**
وەڵام: سیستەمی ئێستا پشتگیری تۆمارکردنی دەستی دەکات. پەیوەندی بە بەڕێوەبەری سیستەمەکەتەوە بکە دەربارەی بژاردەکانی هاوردەکردنی بەردەست بۆ فۆرماتی بانکی تایبەت بە خۆت.

**پرسیار: چۆن مامەڵە لەگەڵ ڕاپۆرتەکانی بانکی پارە جیاوازەکاندا بکەم؟**
وەڵام: هەر هەژمارێکی بانک دەبێت ڕاپۆرتی لە پارەی بنەڕەتی خۆیدا هەبێت. هەڵبژاردەکانی پارەی بیانی بەکاربێنە بۆ بەدواداچوونی بڕی ڕەسەنی مامەڵەکان کاتێک لە پارەی هەژمارەکە جیاوازن.
