# پارەدانەکان: جووڵەی پارە و بەڕێوەبردنی کاش

ئەم ڕێبەرییە گشتگیرە ڕوونیدەکاتەوە چۆن پارەدانەکان کار دەکەن لە سیستمی ژمێریاریدا، کە خۆی دەبینێتەوە لە دروستکردن، بەڕێوەبردن، پشتگیریکردنی فرە-پارە، و هاوتاکردنی بانکی. ئەم ڕێبەرییە نووسراوە بۆ هەموو بەکارهێنەرێک — ژمێریار بێت یان بەکارهێنەری ئاسایی — لەم ڕێبەرییەدا هەوڵدراوە بە نموونەی کردەیی هەنگاوەکان باس بکرێت کە پەیڕەوی لە باشترین شێوازەکانی ژمێریاری دوو-تۆماری دەکات.

---

## پارەدان چییە؟

پارەدان نوێنەرایەتی جووڵەی پارە دەکات لە نێوان کۆمپانیاکەت و هاوبەشێکی بازرگانیدا (کڕیار یان فرۆشیار). پارەدانەکان بنەمان بۆ بەڕێوەبردنی جووڵەی پارە و یەکلاکردنەوەی هەژمارەکانی وەرگرتن/پارەدانەکان.

### جۆرەکانی پارەدان
- **پارەدانی وەرگیراو** — پارەی وەرگیراو لە کڕیاران (هەژمارەکانی وەرگرتن کەم دەکاتەوە)
- **پارەدانی نێردراو** — پارەی نێردراو بۆ فرۆشیاران (هەژمارەکانی پارەدانەکان کەم دەکاتەوە)

### پۆلێنکردنی پارەدان
- **پارەدانە پشتڕاستکراوەکان** — بە پسووڵە دیاریکراوەکان یان پسووڵەی فرۆشیارەکانەوە بەستراوەتەوە بۆ یەکلاکردنەوەی بڕە پارە ماوەکان
- **پێشینەی هاوبەش/قەرز** — پارەدانی سەربەخۆ بۆ پێشینەی کڕیاران یان پێشینەی فرۆشیاران

هەر پارەدانێک ئەمانە لەخۆدەگرێت:
- پەرتووکی ڕۆژانە (هەژماری بانک یان کاش کە پارەکەی تێدا گوازراوەتەوە)
- بەروار و پارەی پارەدان
- بڕ بە هەردوو پارەی پارەدان و پارەی بنەڕەتی کۆمپانیا
- شێوازی پارەدان و ژمارەی بەڵگەی هەڵبژاردەیی
- زانیاری هاوبەش و بەدواداچوونی بارودۆخ

---

## پێداویستیەکانی سیستەم

### ڕێکخستنی پەرتووکی ڕۆژانە
- **پەرتووکی ڕۆژانەی بانک**: لانیکەم یەک پەرتووکی ڕۆژانەی بانک بۆ پرۆسەی پارەدان ڕێکخرابێت
- **شێوازەکانی پارەدان**: شێوازەکانی پارەدان دەبێت بۆ بژاردەکانی پرۆسە کردن ڕێکخرابن
- **دانانی پارە**: پارەکان دەبێت چالاک بن و نرخەکانی ئاڵوگۆڕ تۆمارکرابن
- **تۆمارەکانی هاوبەش**: کڕیار و فرۆشیارەکان دەبێت بۆ بەستنەوەی پارەدان ئامادەبن

### پێشمەرجەکان
1. **دانانی هەژماری بانک**: پەرتووکی ڕۆژانەی بانک بە دروستی لەگەڵ یەکخستنی ژمێریاری ڕێکخرابن
2. **زانیاری هاوبەش**: تۆمارەکانی کڕیار یان فرۆشیار بە وردەکارییەکانی پارەدانەوە دروستکرابن
3. **تۆمارەکانی پسووڵەی کڕیار/پسووڵەی فرۆشیار**: بۆ پارەدانە بەستراوەکان، پسووڵەی کڕیار یان پسووڵەی فرۆشیارەکان دەبێت نێردرابن
4. **مۆڵەتەکانی بەکارهێنەر**: دەستگەیشتنی گونجاو بە تایبەتمەندییەکانی بانک و پارەدان

---

## چۆن لە ڕووکاری بەکارهێنەردا بیدۆزینەوە

بڕۆ بۆ **ژمێریاری → بانک و کاش → پارەدانەکان**

پارەدانەکان لەم بەشانەشدا دەردەکەون:
- **پسووڵەکانی کڕیار**: کرداری "تۆمارکردنی پارەدان" بۆ پسووڵە نێردراوە نەدراوەکان
- **پسووڵەکانی فرۆشیار**: کرداری "تۆمارکردنی پارەدان" بۆ پسووڵە نێردراوە نەدراوەکان
- **هاوتاکردنی بانکی**: پارەدانە پشتڕاستکراوەکان بۆ هاوتاکردن لەگەڵ ڕاپۆرتەکانی بانک بەردەستن
- **داشبۆرد**: چالاکییەکانی پارەدانی تازە و کورتەی جووڵەی پارە

**تێبینی**: دوگمەی یارمەتی/بەڵگەنامە لە سەرەوە ئەم ڕێبەرییە دەکاتەوە.

---

## ڕەوتی کاری بارودۆخی پارەدان

پارەدانەکان پەیڕەوی لە ڕەوتی کارێکی کۆنترۆڵکراو دەکەن بۆ دڵنیابوون لە دروستی داتاکان (زانیارییەکان):

### ڕەشنووس
- پارەدانی تازە دروستکراو، بە تەواوی دەستکاریدەکرێت
- دەتوانرێت دەستکاری یان بسڕدرێتەوە
- هیچ کاریگەرییەکی ژمێریاری دروست ناکات جارێ
- هیچ تۆمارێکی ڕۆژانە دروست ناکات جارێ

### پشتڕاستکراو
- پارەدان کۆتایی پێهێنراوە و نێردراوە بۆ ژمێریاری
- تۆماری ڕۆژانە خۆکارانە دروستکراوە
- گۆڕانکاری وەرناگرێت بۆ پاراستنی ڕێڕەوی پشکنین
- ناتوانرێت دەستکاری یان بسڕدرێتەوە
- نرخەکانی ئاڵوگۆڕ لە کاتی پشتڕاستکردنەوەدا قوفڵ کراون

### هاوتاکراو
- پارەدان لەگەڵ هێڵی ڕاپۆرتی بانک هاوتا کراوە
- پشتڕاستی دەکاتەوە کە مامەڵەی بانکی ڕاستەقینە ڕوویداوە
- تۆماری ڕۆژانەی هاوسەنگی بانکی زیادە دروستکراوە
- دۆخی کۆتاییە لە ڕەوتی کاری ئاساییدا

### هەڵوەشێنراوەتەوە
- پارەدانی پشتڕاستکراو کە گەڕێندراوەتەوە
- تۆماری ڕۆژانەی پێچەوانە دروستکراوە
- ڕێڕەوی پشکنینی تەواو دەپارێزێت
- ناتوانرێت دووبارە هەڵوەشێنرێتەوە کاتێک هەڵوەشێنراوەتەوە

**گرنگ**: تەنها پارەدانەکانی دۆخی ڕەشنووس دەتوانرێن دەستکاری یان بسڕدرێنەوە. پارەدانە پشتڕاستکراوەکان پێویستیان بە ڕێکارەکانی گەڕاندنەوەی گونجاو هەیە بۆ پاراستنی دروستی ژمێریاری.

---

## شێوازەکانی پارەدان

سیستەم پشتگیری چەندین شێوازی پارەدان دەکات:

- **دەستی** — تۆمارکردنی دەستی ئاسایی
- **چێک** — پارەدان لە ڕێگەی چەکەوە
- **گواستنەوەی بانکی** — گواستنەوەی بانکی ئەلکترۆنی
- **کرێدت کارت** — مامەڵەکانی کرێدت کارت
- **دێبت کارت** — مامەڵەکانی دێبت کارت
- **کاش** — پارەدانی کاشی فیزیایی
- **گواستنەوەی پارەی ئەلکترۆنی** — گواستنەوەی پارەی ئەلکترۆنی نێودەوڵەتی
- **ACH** — گواستنەوەی خانووی پاککردنەوەی خۆکار (Automated Clearing House)
- **SEPA** — گواستنەوەی ناوچەی پارەدانی یۆرۆی تاک (Single Euro Payments Area)
- **پارەدانی ئۆنلاین** — پلاتفۆرمەکانی پارەدانی دیجیتاڵی

## دروستکردنی پارەدانەکان

### ۱) پارەدانی سەربەخۆ لە لیستی پارەدانەکانەوە

ئەم هەنگاوانە بەکاربێنە بۆ جووڵەی پارە کە ڕاستەوخۆ بە پسووڵەی کڕیار/پسووڵەی فرۆشیارەکانەوە نەبەستراوەتەوە. بۆ نمونە (پێشینەی کڕیاران، پێشینەی فرۆشیاران، یان پارەدانە دەستبەجێکان کە دواتر دەبەسترێنەوە).

**هەنگاوەکان:**
1. بڕۆ بۆ **ژمێریاری → بانک و کاش → پارەدانەکان**
2. کلیک لە **دروستکردن** بکە
3. فۆرمی پارەدان پڕ بکەرەوە:
    - **جۆری پارەدان**: وەرگرتن (وەرگیراو) یان ناردن (نێردراو) هەڵبژێرە
    - **هاوبەش**: کڕیار یان فرۆشیار هەڵبژێرە
    - **پەرتووکی ڕۆژانە**: هەژماری بانک یان کاش هەڵبژێرە
    - **بەرواری پارەدان**: کەی پارەکە گواستراوەتەوە
    - **پارە و بڕ**: جۆری پارە و بڕی پارەدان دیاریبکە
    - **شێوازی پارەدان**: لە شێوازە بەردەستەکان هەڵبژێرە
    - **ژمارەی بەڵگە**: شرۆڤەیەک بنوسە (هەڵبژاردەییە) یان ژمارەی بەڵگەی بانک
4. **پاشەکەوتکردن** بۆ دروستکردن وەک ڕەشنووس
5. **پشتڕاستکردنەوە** کاتێک ئامادە بوو بۆ ناردن بۆ ژمێریاری

**ئەنجام:**
- پارەدانی ڕەشنووس دروست دەکرێت و دەستکاریدەکرێت
- لە کاتی پشتڕاستکردنەوەدا، تۆماری ڕۆژانە خۆکارانە دەنێردرێت
- پارەدان بۆ یەکلاکردنەوە یان هاوتاکردنی بانکی بەردەستە

### ۲) تۆمارکردنی پارەدان لەسەر پسووڵە (پارەدانی کڕیار)

ئەم هەنگاوانە بەکاربێنە بۆ وەرگرتنی پارە بۆ پسووڵەکانی کڕیاری نێردراو.

**پێشمەرجەکان:**
- پسووڵە دەبێت نێردرابێت (نەک ڕەشنووس)
- بڕی ماوەی شایستە دەبێت بەردەستبێت

**هەنگاوەکان:**
1. پسووڵەی نێردراو بکەرەوە
2. کلیک لە **تۆمارکردنی پارەدان** بکە (ئایکۆنی پارەی کاغەزی)
3. دیالۆگی پارەدان پڕ بکەرەوە:
    - **پەرتووکی ڕۆژانە**: بە شێوەی ئاسایی هەژماری بانکییە سەرەکیەکەت
    - **بەرواری پارەدان**: کەی پارەکە وەرگیراوە
    - **بڕ**: بە شێوەی خۆکارانە بڕی ماوە نیشان دەدات؛ بۆ پارەدانی بەش دەستکاری بکە
    - **ژمارەی بەڵگە**: ژمارەی بەڵگەی بانک یان شرۆڤە (هەڵبژاردەییە)
4. **پشتڕاستکردنەوە** بۆ دروستکردن و ناردنی پارەدان

**چی ڕوودەدات:**
- پارەدانی وەرگیراو دروست دەکرێت و دەستبەجێ پشتڕاست دەکرێتەوە
- پارەدان بە پسووڵەوە دەبەسترێتەوە، بڕی شایستە کەم دەکاتەوە
- بارودۆخی پسووڵە نوێ دەکرێتەوە (بەشێکی پارەدراو یان هەمووی پارەدراوە)
- تۆماری ڕۆژانە خۆکارانە دەنێردرێت
- پارەدان بۆ هاوتاکردنی بانکی بەردەستە

### ۳) تۆمارکردنی پارەدان لەسەر پسووڵەی فرۆشیار (پارەدانی فرۆشیار)

ئەم هەنگاوانە بەکاربێنە بۆ پارەدانی پسووڵەکانی فرۆشیاری نێردراو.

**پێشمەرجەکان:**
- پسووڵەی فرۆشیار دەبێت نێردرابێت (نەک ڕەشنووس)
- بڕی ماوەی شایستە دەبێت هەبێت

**هەنگاوەکان:**
1. پسووڵەی فرۆشیاری نێردراو بکەوە
2. کلیک لە **تۆمارکردنی پارەدان** بکە (ئایکۆنی پارەی کاغەزی)
3. دیالۆگی پارەدان پڕ بکەرەوە:
    - **پەرتووکی ڕۆژانە**: بە شێوەی ئاسایی هەژماری بانکییە سەرەکیەکەت
    - **بەرواری پارەدان**: کەی پارەکە نێردراوە
    - **بڕ**: بە شێوەی خۆکارانە بڕی ماوە نیشان دەدات؛ بۆ پارەدانی بەش دەستکاری بکە
    - **ژمارەی بەڵگە**: ژمارەی بەڵگەی بانک یان شرۆڤە (هەڵبژاردەییە)
4. **پشتڕاستکردنەوە** بۆ دروستکردن و ناردنی پارەدان

**چی ڕوودەدات:**
- پارەدانی نێردراو دروست دەکرێت و دەستبەجێ پشتڕاست دەکرێتەوە
- پارەدان بە پسووڵەی فرۆشیارەوە دەبەسترێتەوە، بڕی شایستە کەم دەکاتەوە
- بارودۆخی پسووڵەی فرۆشیار نوێ دەکرێتەوە (بەشێکی پارەدراو یان هەمووی پارەدراوە)
- تۆماری ڕۆژانە خۆکارانە دەنێردرێت
- پارەدان بۆ هاوتاکردنی بانکی بەردەستە

## پشتڕاستکردنەوە و بەڕێوەبردنی پارەدان

### پشتڕاستکردنەوەی پارەدانەکان

**دۆخی ڕەشنووس** دەتوانرێن پشتڕاست بکرێنەوە و بنێردرێت بۆ ژمێریاری:

1. پارەدانی ڕەشنووس بکەوە
2. کلیک لە **پشتڕاستکردنەوەی پارەدان** بکە
3. کردارەکە پشتڕاست بکەرەوە کاتێک داوا دەکات

**دوای پشتڕاستکردنەوە:**
- تۆماری ڕۆژانە خۆکارانە دروست دەکرێت
- نرخەکانی ئاڵوگۆڕ قوفڵ دەکرێن (بۆ فرە-پارە)
- پارەدانەکە گۆڕانکاری وەرناگرێت
- بارودۆخ دەگۆڕێت بۆ پشتڕاستکراوە

### سنووردارکردنی دەستکاری

- **دۆخی ڕەشنووس**: بە تەواوی دەستکاریدەکرێن و دەتوانرێت بسڕدرێنەوە
- **پارەدانە پشتڕاستکراوەکان**: گۆڕانکاری وەرناگرن، ناتوانرێن دەستکاری بکرێن یان بسڕدرێنەوە
- **پارەدانە هاوتاکراوەکان**: دۆخی کۆتایی، هیچ گۆڕانکارییەک ڕێگەپێدراو نییە

### یاساکانی سڕینەوە

- **دۆخی ڕەشنووس**: دەتوانرێت بە سەلامەتی بسڕدرێنەوە
- **پارەدانە پشتڕاستکراوە/هاوتاکراوەکان**: ناتوانرێت بسڕدرێنەوە
- هەڵوەشاندنەوە بەکاربێنە بۆ پارەدانە پشتڕاستکراوەکان کە پێویستیان بە گەڕاندنەوە هەیە

## پارەدانە فرە-پارەکان

سیستەم پشتگیری پارەدان بە هەر پارەیەکی ڕێکخراو دەکات لەگەڵ گۆڕینی خۆکارانە:

### مامەڵەکردنی پارە
- **پارەی پارەدان**: ئەو پارەیەی کە پارەدانی پێ کراوە
- **پارەی کۆمپانیا**: پارەی بنەڕەتی کۆمپانیاکەت بۆ ڕاپۆرتکردن
- **نرخی ئاڵوگۆڕ**: خۆکارانە لە کاتی پشتڕاستکردنەوەی پارەداندا وەردەگیرێت
- **ئاڵوگۆڕ**: گۆڕینی خۆکارانە بۆ پارەی کۆمپانیا

### بەڕێوەبردنی نرخی ئاڵوگۆڕ
- نرخەکان لە بەرواری پارەداندا وەردەگیرێن
- لەگەڵ پشتڕاستکردنەوە قوفڵ دەکرێت بۆ ڕێگریکردن لە گۆڕانکاری
- بۆ ژماردنی قازانج/زیانی بەدەستهاتوو بەکاردەهێنرێن
- لە وردەکارییەکانی پارەداندا پیشاندەدرێن بۆ ڕێڕەوی پشکنین

### قازانج/زیانی ئاڵوگۆڕی پارەی بەدەستهاتوو
کاتێک پسووڵەی کڕیار/پسووڵەی فرۆشیاری پارەی بیانی یەکلا دەکرێنەوە:
- سیستەم جیاوازی نێوان نرخی پسووڵە و نرخی پارەدان دەژمێرێت
- تۆماری ڕۆژانەی خۆکارانە بۆ قازانج/زیانی بەدەستهاتوو
- نێردراوە بۆ هەژمارەکانی قازانج/زیانی ئاڵوگۆڕی پارەی ڕێکخراو
- ڕاپۆرتی دروستی دارایی دەپارێزێت

**نموونە**:
- پسووڵە: $1,000 دۆلاری ئەمریکی بە نرخی 1,300 دیناری عێراقی/دۆلاری ئەمریکی = 1,300,000 دیناری عێراقی
- پارەدان: $1,000 دۆلاری ئەمریکی بە نرخی 1,310 دیناری عێراقی/دۆلاری ئەمریکی = 1,310,000 دیناری عێراقی
- زیانی بەدەستهاتوو: 10,000 دیناری عێراقی نێردراوە بۆ هەژماری زیانی ئاڵوگۆڕی پارە

## پەیوەندییەکانی پارەدان و بەستەرەکانی بەڵگەنامە

### بەڵگەنامە بەستراوەکان
پارەدانەکان دەتوانرێت بە چەندین بەڵگەنامەوە ببەسترێنەوە:
- **پسووڵەکانی کڕیار**: یەکلاکردنەوەی پارەدانی کڕیاران
- **پسووڵەکانی فرۆشیار**: یەکلاکردنەوەی پارەدانی فرۆشیاران
- **بڕی بەکارهاتوو**: بەدواداچوونی بڕە دیاریکراوەکان کە بۆ هەر بەڵگەنامەیەک تەرخانکراون

### بینینی پەیوەندییەکان
هەر پارەدانێک زانیاری پەیوەندیدار پیشان دەدات:
- **تابی پسووڵەکان**: هەموو پسووڵەکانی کڕیاری بەستراو
- **تابی پسووڵەکانی فرۆشیار**: هەموو پسووڵەکانی فرۆشیاری بەستراو
- **تابی تۆماری ڕۆژانە**: هەموو تۆمارە ژمێریارییەکان
- **تابی هێڵەکانی ڕاپۆرتی بانک**: مێژووی هاوتای بانکی

### یەکلاکردنەوەی بەڵگەنامە
- پارەدانەکان دەتوانن چەندین بەڵگەنامە یەکلا بکەنەوە
- پشتگیری یەکلاکردنەوەی بەش دەکات
- بڕە ماوەکان خۆکارانە بەدواداچوونیان بۆ دەکرێت
- مێژووی یەکلاکردنەوە بۆ پشکنین دەپارێزرێت

---

## هاوتاکردنی بانکی

### پرۆسەی هاوتاکردنی بانکی
1. ڕاپۆرتەکانی بانک هاوردە بکە یان دروست بکە
2. بڕۆ بۆ **ژمێریاری → بانک و کاش → ڕاپۆرتەکانی هەژماری بانک**
3. ڕاپۆرتەکە بکەوە و کلیک لە **هاوتاکردنی بانکی** بکە
4. پارەدانە پشتڕاستکراوەکان لەگەڵ هێڵەکانی ڕاپۆرتی بانک هاوتا بکە
5. سیستەم بارودۆخی پارەدان بۆ هاوتاکراو نوێ دەکاتەوە

### کاریگەرییەکانی هاوتاکردنی بانکی
- بارودۆخی پارەدان دەگۆڕێت بۆ هاوتاکراو
- هێڵی ڕاپۆرتی بانک وەک هاوتاکراو نیشان دەکرێت
- تۆماری ڕۆژانەی هاوتاکردنی بانکی زیادە دروست دەکرێت
- پشتڕاستی دەکاتەوە کە مامەڵەی بانکی ڕاستەقینە ڕوویداوە

### پێداویستییەکانی هاوتاکردنی بانکی
- پارەدانەکان دەبێت پشتڕاستکرابن پێش هاوتاکردنی بانکی
- بڕی پارەکان دەبێت لە نێوان پارەدان و هێڵی ڕاپۆرتی بانکدا یەکبگرنەوە
- گۆڕینی پارە خۆکارانە مامەڵەی پێوە دەکرێت
- چەندین پارەدان دەتوانرێت لەگەڵ یەک هێڵی ڕاپۆرتی بانک هاوتا بکرێت

---

## باشترین شێوازەکان و ئامۆژگارییەکان

### بەڕێوەبردنی پارەدان
- **بە ووردی پشتڕاستی بکەرەوە**: پارەدانە پشتڕاستکراوەکان ناتوانرێن دەستکاری بکرێن
- **ژمارەی بەڵگە بەکاربێنە**: ژمارەی بەڵگەی بانک زیاد بکە بۆ ئاسانکاری هاوتاکردنی بانکی
- **هاوتاکردنی بانکی بەردەوام**: پارەدانەکانت لەگەڵ ڕاپۆرتەکانی بانک یەکبخە
- **پارەدانی بەش**: بۆ وەرگرتنی/پێدانی قیستەکان بەکاربێنە

### باشترین شێوازەکانی فرە-پارە
- **چاودێری نرخی ئاڵوگۆڕی پارە بکە**: گۆڕانکارییە گەورەکانی نرخ کاریگەرییان لەسەر جووڵەی پارە هەیە
- **مەترسی پارە کەم بکەرەوە**: بیر لە گرێبەستی پێشوەختە بکەرەوە بۆ بڕە گەورەکان
- **پێداچوونەوەی قازانج/زیان**: کاریگەری قازانج/زیانی ئاڵوگۆڕی پارە لەسەر قازانج چاودێری بکە

### پشکنین و پابەندبوون
- **بەڵگەنامە بپارێزە**: بەڵگەنامە پشتگیریکەرەکانی هەموو پارەدانەکان بپارێزە
- **پێداچوونەوەی تۆماری ڕۆژانە**: کاریگەری ژمێریاری پارەدانەکان پشتڕاست بکەرەوە
- **پابەندبوون بە بەرواری قوفڵکراو**: ڕەچاوی بەروارەکانی قوفڵکراوی ماوەی ژمێریاری بکە
- **جیاکردنەوەی ئەرکەکان**: دروستکردن و پشتڕاستکردنەوەی پارەدانەکان جیا بکەرەوە

---

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**پرسیار: ناتوانرێت پارەدانی پشتڕاستکراو دەستکاری بکرێت**

وەڵام: پارەدانە پشتڕاستکراوەکان گۆڕانکاری وەرناگرن. ئەگەر گۆڕانکاری پێویست بوو، تۆماری گەڕاندنەوە دروست بکە.

**پرسیار: پارەدان لە هاوتاکردنی بانکیدا دەرناکەوێت**

وەڵام: دڵنیابەرەوە پارەدانەکە پشتڕاستکراوەتەوە و لەگەڵ  جۆری پارە/بڕی ڕاپۆرتەکە یەکدەگرێتەوە.

**پرسیار: نرخی ئاڵوگۆڕی پارە هەڵە دەردەکەوێت**

وەڵام: نرخەکانی ئاڵوگۆڕ لە کاتی پشتڕاستکردنەوەدا قوفڵ دەکرێن. سەرچاوەی نرخ و بەروار بپشکنە.

**پرسیار: ناتوانرێت پارەدان بسڕدرێتەوە**

وەڵام: تەنها پارەدانەکانی دۆخی ڕەشنووس دەتوانرێت بسڕدرێنەوە. بۆ پارەدانە پشتڕاستکراوەکان هەڵوەشاندنەوە بەکاربێنە.

### پەیامەکانی هەڵە

**"تەنها پارەدانەکانی ڕەشنووس دەتوانرێت پشتڕاست بکرێنەوە"**
- پارەدانەکە پێشتر پشتڕاستکراوەتەوە یان لە دۆخی هەڵەدایە
- وێبگەڕەکەت نوێ بکەرەوە و بارودۆخی ئێستا بپشکنە

**"بەرواری پارەدان لە ماوەی قوفڵکراودایە"**
- بەرواری پارەدان دەکەوێتە ناو ماوەی ژمێریاری قوفڵکراو
- بەروار بگۆڕە یان داوای کردنەوەی ماوەکە لە بەڕێوەبەری سیستەم بکە

**"بڕی پارەی ماوەی کەم"**
- هەوڵ دەدەیت زیاتر لە بڕی ماوەی بەڵگەنامەکە بەکاربهێنیت
- بارودۆخی بەڵگەنامە و پارەدانەکانی پێشوو بپشکنە

## پرسیارە دووبارەبووەکان

**پرسیار: دەتوانم پارەدانی پشتڕاستکراو بسڕمەوە؟**

وەڵام: نەخێر. بەڵگەنامە داراییەکان لە کاتی پشتڕاستکردنەوەدا گۆڕانکاری وەرناگرن بۆ پاراستنی ڕێڕەوی پشکنین. بۆ گەڕاندنەوەکان هەڵوەشاندنەوە بەکاربێنە.

**پرسیار: دەتوانم دەتوانم تەنها بەشێکی پارەکە بدەم؟**

وەڵام: بەڵێ. بڕێکی بچووکتر بنووسە کاتێک پارەدان تۆمار دەکەیت. بەڵگەنامەکە بەشێکی دراو دەمێنێتەوە لەگەڵ هاوسەنگی نوێکراوە.

**پرسیار: پارەی بیانی چۆن کار دەکات؟**

وەڵام: بڕەکان لە هەردوو پارەی پارەدان و پارەی کۆمپانیا هەڵدەگرین. قازانج/زیانی ئاڵوگۆڕی پارە خۆکارانە دەژمێردرێت و دەنێردرێت کاتێک بەڵگەنامەکان یەکلا دەکرێنەوە.

**پرسیار: جیاوازی نێوان پارەدانی یەکلاکردنەوە و پێشینە چییە؟**

وەڵام: پارەدانەکانی یەکلاکردنەوە بە پسووڵەی کڕیار/پسووڵەی فرۆشیاری دیاریکراوەوە بەستراونەتەوە. پارەدانەکانی پێشینە سەربەخۆن و دەتوانرێن بۆ بەڵگەنامەکانی داهاتوو بەکاربهێنرێن.

**پرسیار: دەتوانم بە یەک پارەدان چەند پسووڵەیەک یەکلا بکاتەوە؟**

وەڵام: بەڵێ. بەستەرەکانی بەڵگەنامە بەکاربێنە بۆ تەرخانکردنی بڕی پارەدان بەسەر چەند پسووڵە یان پسووڵەی فرۆشیاردا.

**پرسیار: چۆن مامەڵە لەگەڵ نرخی تێچووی پارەداندا بکەم؟**

وەڵام: تێچووی مامەڵە داراییەکان بە جیا لە ڕێی ڕاپۆرتەکانی بانک یان تۆماری ڕۆژانەی گشتییەوە تۆمار بکە. بڕی پارەدانەکان دەبێت ڕەنگدانەوەی بڕی پارەی وەرگیراو/نێردراو نیشان بدەن.

## فەرهەنگ

- **هاوبەش**: کڕیار یان فرۆشیار لە سیستەمەکەدا
- **پەرتووکی ڕۆژانە**: هەژماری بانک یان کاش کە مامەڵەکانی تێدا تۆمار دەکرێن
- **یەکلاکردنەوە**: بەکارهێنانی پارەدان بۆ پسووڵە یان پسووڵەی فرۆشیاری دیاریکراو
- **هاوتاکردنی بانکی**: هاوتاکردنی پارەدانەکانی سیستەم لەگەڵ هێڵەکانی ڕاپۆرتی بانک
- **بەستەری بەڵگەنامە**: پەیوەندی نێوان پارەدان و پسووڵەی کڕیار/پسووڵەی فرۆشیار لەگەڵ بڕی بەکارهاتوو
- **نرخی ئاڵوگۆڕی پارە**: نرخی گۆڕینی پارە کە لە کاتی پشتڕاستکردنەوەی پارەداندا قوفڵ دەکرێت
- **قازانج/زیانی بەدەستهاتوو**: جیاوازی نرخی ئاڵوگۆڕی پارە کاتێک بەڵگەنامەکانی پارەی بیانی یەکلا دەکرێنەوە
