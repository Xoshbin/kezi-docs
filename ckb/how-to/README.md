# How-to Guides

How-to guides are task-oriented recipes that solve specific problems. They assume the user knows what they want to achieve but doesn't know how to do it.

## Structure
- **Goal**: Complete a task
- **Tone**: Helper, Colleague ("How to X")
- **Content**: A specific problem and its solution steps.

## Examples
- "How to cancel a paid invoice"
- "How to add a custom tax rate"
