# نوێکردنەوەی نرخی پارە: بەڕێوەبردنی جیاوازی نرخی ئاڵوگۆڕ

ئەم ڕێنماییە ڕوونی دەکاتەوە چۆن نوێکردنەوەی نرخی پارە بەکاربهێنیت بۆ ڕاستکردنەوەی باڵانسی پارە بیانییەکانت بەگوێرەی نرخی ئاڵوگۆڕی ئێستا. جا کۆتایی مانگ بێت یان کۆتایی ساڵ، ئەم پرۆسەیە دڵنیایی دەدات کە ڕاپۆرتە داراییەکانت بە ڕاستی نرخی سامان و قەرزەکانی پارەی بیانی نیشان دەدەن.

---

## نوێکردنەوەی نرخی پارە چییە؟

**نوێکردنەوەی نرخی پارە** نرخی پەرتووکی باڵانسی پارە بیانی ڕاستدەکاتەوە بەگوێرەی نرخی ئاڵوگۆڕی ئێستا. کاتێک نرخی ئاڵوگۆڕ دەگۆڕێت لەنێوان کاتی تۆمارکردنی مامەڵە و کۆتایی ماوە، نرخەکە بە پارەی بنەڕەتیت دەگۆڕێت—ئەگەرچی بڕی پارە بیانییەکە هەمان دەمێنێتەوە.

**بۆچی گرنگە؟**

| هۆکار | نموونە |
|--------|---------|
| **لیستەی باڵانسی دروست** | قەرزی وەرگرتنی ١٠,٠٠٠ دۆلار کاتی تۆمارکردن بەنرخی ١٤.٥ ملیۆن دینار بوو، بەڵام ئێستا ١٤.٧ ملیۆن دینارە |
| **پابەندبوون بە یاسا** | IFRS و زۆربەی ستانداردەکانی ژمێریاری نوێکردنەوەی کۆتایی ماوە داوا دەکەن |
| **ڕوونی دارایی** | قازانج/زیانی نەڕۆشتوو ببینە پێش ئەوەی ڕۆشتوو بن |
| **داخستنی مانگ** | هەنگاوێکی پێویستە پێش کۆتاییکردنی ڕاپۆرتە داراییەکان |

**کاریگەری ژمێریاری**: ڕاستکردنەوەی نوێکردنەوە قازانج یان زیانی نەڕۆشتووی ئاڵوگۆڕ لە لیستی قازانج و زیان تۆمار دەکات، لەگەڵ ڕاستکردنەوەی نرخی پەرتووکی هەژمارەکانی پارە بیانی.

---

## لەکوێ دەتوانی بیدۆزیتەوە

بڕۆ بۆ: **ژمێریاری ← نوێکردنەوەی نرخی پارە**

لیستی هەموو نوێکردنەوەکان دەبینیت لەگەڵ دۆخ، ڕێکەوت، و بڕی ڕاستکردنەوەی تاقە.

> [!TIP]
> نوێکردنەوەی نرخی پارە لە کۆتایی هەر ماوەی ژمێریاریدا بەڕێوەببە (مانگانە، سێمانگانەوە، یان ساڵانە) پێش دروستکردنی ڕاپۆرتە داراییەکان.

---

## کاروبار و ڕەوتی نوێکردنەوە

نوێکردنەوەکەت لەم قۆناغانەدا تێدەپەڕێت:

┌─────────┐      ┌───────────┐      ┌───────────────┐
│ ڕەشنووس │ ──▶ │ نێردراوە │ ──▶ │پێچەوانەکراوە │
└─────────┘      └───────────┘      └───────────────┘
    📝               ✅                   ↩️

### 📝 ڕەشنووس
- سیستەم قازانج/زیانی نەڕۆشتوو حیساب دەکات
- دەتوانی پێداچوونەوە بکەیت و گۆڕانکاری بکەیت
- هیچ کاریگەری ژمێریاری نییە هێشتا
- سەلامەتە بۆ سڕینەوە ئەگەر پێویست نەبوو

### ✅ نێردراوە
- تۆماری ڕۆژانە دروستکراوە
- قازانج/زیان لە لیستی قازانج و زیان تۆمارکراوە
- هەژمارەکانی لیستەی باڵانس ڕاستکراوەتەوە
- دەتوانرێت پێچەوانە بکرێتەوە ئەگەر پێویست بوو

### ↩️ پێچەوانەکراوەتەوە
- تۆماری ڕۆژانەی پێچەوانە دروستکراوە
- بەسود ئەگەر پێویستت بە دووبارە بەڕێوەبردن بوو بە نرخی جیاوازەوە

---

## چۆن حیسابەکە کاردەکات

بۆ هەر باڵانسێکی پارە بیانی، سیستەم ئەمانە حیساب دەکات:

```
نرخی پەرتووک       = باڵانسی بیانی × نرخی تاریخی
بڕی نوێکراوەتەوە   = باڵانسی بیانی × نرخی ئێستا
ڕاستکردنەوە       = بڕی نوێکراوەتەوە - نرخی پەرتووک
```

**نموونە**: ١٠,٠٠٠ دۆلارت هەیە لە قەرزی وەرگرتن

| فاکتەر | نرخ |
|--------|-------|
| باڵانسی بیانی | ١٠,٠٠٠ دۆلار |
| نرخی تاریخی | ١ دۆلار = ١,٤٥٠ دینار |
| نرخی پەرتووک | ١٤,٥٠٠,٠٠٠ دینار |
| نرخی ئێستا | ١ دۆلار = ١,٤٧٠ دینار |
| بڕی نوێکراوەتەوە | ١٤,٧٠٠,٠٠٠ دینار |
| **ڕاستکردنەوە (قازانج)** | **+٢٠٠,٠٠٠ دینار** |

---

## کام هەژمارەکان کاریگەر دەبن؟

نوێکردنەوەی نرخی پارە بە تایبەتی هەژمارەکانی باڵانسی پارە بیانی کاریگەر دەکات:

| جۆری هەژمار | نموونە هەژمار |
|--------------|------------------|
| **قەرزی وەرگرتن** | پسووڵەی فرۆشتن بە دۆلار، یۆڕۆ، هتد. |
| **قەرزی پێدان** | پسووڵەی کڕین بە پارە بیانی |
| **هەژماری بانک** | هەژماری بانکی پارە بیانی |
| **هەژماری قەرز** | قەرزی پارە بیانی کە دراوە یان وەرگیراوە |

> [!IMPORTANT]
> تەنها هەژمارەکان کە مامەڵەی پارە بیانییان هەیە (جیاواز لە پارەی بنەڕەتیت) ڕاستکردنەوەی نوێکردنەوە نیشان دەدەن.

---

## دروستکردنی نوێکردنەوەی نرخ

### هەنگاوی ١: دەستپێکردن

١. بڕۆ بۆ **ژمێریاری ← نوێکردنەوەی نرخی پارە**
٢. کلیک بکە لەسەر **دروستکردنی نوێکردنەوەی نرخ**

### هەنگاوی ٢: پڕکردنەوەی زانیاریەکان

| خانە | چی بنووسیت | نموونە |
|-------|---------------|---------|
| **ڕێکەوتی نوێکردنەوە** | ڕۆژی گەڕانەوەی نرخ | کۆتا ڕۆژی مانگ (٣١ی کانونی یەکەم) |
| **وەسف** | تێبینی ئارەزوومەندانە | "نوێکردنەوەی کۆتایی مانگی کانونی یەکەم ٢٠٢٥" |

### هەنگاوی ٣: پێداچوونەوەی هێڵە حیسابکراوەکان

سیستەم بە شێوەی خۆکار:
- هەموو هەژمارەکان دەدۆزێتەوە کە باڵانسی پارە بیانییان هەیە
- نرخی ئاڵوگۆڕی ئێستا بۆ ڕێکەوتی نوێکردنەوە دەگەڕێنێت
- ڕاستکردنەوە حیساب دەکات بۆ هەر تەواوکردنێکی هەژمار/پارە/هاوبەش

دەبینیت بۆ هەر هێڵێک:
- **هەژمار**: هەژماری نوێکراوەتەوە
- **پارە**: پارە بیانییەکە
- **باڵانسی بیانی**: بڕ بە پارە بیانی
- **نرخی تاریخی**: ناوەندی نرخ کاتی تۆمارکردنی مامەڵەکان
- **نرخی ئێستا**: نرخ لە ڕێکەوتی نوێکردنەوە
- **نرخی پەرتووک**: نرخی پەرتووکی ئێستا بە پارەی بنەڕەتی
- **بڕی نوێکراوەتەوە**: نرخی نوێ بە نرخی ئێستا
- **ڕاستکردنەوە**: جیاوازی (قازانج یان زیان)

### هەنگاوی ٤: پێداچوونەوەی کۆیەکان

پێش ناردن، کۆیەکان بپشکنە:
- **کۆی قازانج**: کۆی هەموو ڕاستکردنەوە ئەرێنییەکان
- **کۆی زیان**: کۆی هەموو ڕاستکردنەوە نەرێنییەکان
- **ڕاستکردنەوەی تاقە**: قازانج یان زیانی گشتی

### هەنگاوی ٥: ناردنی نوێکردنەوە

کاتێک ڕازیبوویت، کلیک بکە لەسەر **ناردن** بۆ:
- دروستکردنی تۆماری ڕۆژانە
- داخستنی تۆماری نوێکردنەوە
- نوێکردنەوەی ڕاپۆرتە داراییەکان

---

## چی ڕوودەدات لەپشت پەردە

کاتێک نوێکردنەوەیەک دەنێریت بە قازانجی تاقەی ٢٠٠,٠٠٠ دینار لەسەر قەرزی وەرگرتن:

```
┌─────────────────────────────────────────────────────────────┐
│  قەرزدار. قەرزی وەرگرتن         200,000 دینار               │
│      قەرزدەر. قازانجی ئاڵوگۆڕی نەڕۆشتوو       200,000 دینار │
└─────────────────────────────────────────────────────────────┘
```

**بە کوردی ئاسان**: نرخی پەرتووکی قەرزەکانمان زیادەدەکەین بۆ هاوتابوون بە نرخی ئاڵوگۆڕی ئێستا، و قازانجەکە لە لیستی داهاتمان تۆمار دەکەین.

بۆ زیانی تاقەی ١٥٠,٠٠٠ دینار لەسەر قەرزی پێدان:

```
┌─────────────────────────────────────────────────────────────┐
│  قەرزدار. زیانی ئاڵوگۆڕی نەڕۆشتوو    150,000 دینار          │
│      قەرزدەر. قەرزی پێدان                   150,000 دینار   │
└─────────────────────────────────────────────────────────────┘
```

## باڵەخانەی باو

### باڵەخانەی ١: داخستنی مانگ

**دۆخەکە**: ٣١ی کانونی دووەمە و پێویستت بە داخستنی مانگ. کۆمپانیاکەت قەرزی وەرگرتنی دۆلار و قەرزی پێدانی یۆڕۆ هەیە.

**چی بکەیت**:

١. دڵنیابە کە نرخی ئاڵوگۆڕ بۆ ٣١ی کانونی دووەم لە سیستەمدا تۆمارکراوە
٢. نوێکردنەوەی نرخی نوێ دروستبکە
٣. **ڕێکەوتی نوێکردنەوە** بخە سەر ٣١ی کانونی دووەم
٤. ڕاستکردنەوە حیسابکراوەکان پێداچوونەوە بکە
٥. نوێکردنەوەکە **بنێرە**
٦. ڕاپۆرتە داراییەکانی کۆتایی مانگ دروستبکە

**ئەنجام**:
- لیستەی باڵانس کاڵا و قەرزەکانی پارە بیانی بە نرخی ئێستا نیشان دەدات
- لیستی قازانج و زیان قازانج/زیانی ئاڵوگۆڕی نەڕۆشتوو لەخۆدەگرێت
- پێداچونەوەی باڵانس ئامادەیە بۆ داخستنی ماوە

### باڵەخانەی ٢: ئامادەکاری پشکنینی کۆتایی ساڵ

**دۆخەکە**: پشکنەرەکان پێویستیان بە ڕاستکردنەوەی نرخاندنی پارە بیانی بۆ کۆتایی ساڵ هەیە.

**چی بکەیت**:

١. نرخی ئاڵوگۆڕی کۆتایی ساڵ بپشکنەوە و بەڵگەبکە
٢. نوێکردنەوە دروستبکە بۆ ٣١ی کانونی یەکەم
٣. ڕاپۆرتی نوێکردنەوە هەناردە بکە یان چاپی بکە کە نیشان دەدات:
   - هەر باڵانسێکی پارە بیانی
   - نرخی تاریخی و ئێستا
   - حیسابی ڕاستکردنەوەکان
٤. نوێکردنەوەکە **بنێرە**
٥. ڕاپۆرت و تۆماری ڕۆژانە پێشکەشی پشکنەرەکان بکە

## ڕێنمایی باش

### 📅 کاتی گونجاو
- **لە کۆتایی ماوەدا نوێبکەرەوە**: هەمیشە پێش داخستنی پەرتووکەکان
- **نرخەکان بەڵگەبکە**: نرخی ئاڵوگۆڕ تۆمارکراوبێت بۆ هەر ڕێکەوتی نوێکردنەوە
- **کاتی یەکسان**: هەمان ڕۆژ بەکاربهێنە لە هەر ماوەدا (وەک کۆتا ڕۆژی کاری)

### ✅ ڕاستی
- **نرخەکان بپشکنە**: نرخی ئاڵوگۆڕ دووبارە بپشکنە پێش ناردن
- **ڕاستکردنەوەی گەورە پێداچوونەوەبکە**: قازانج/زیانی زۆر لێکۆڵینەوەبکە
- **یەکەم هاوتاکردن**: دڵنیابە کە هاوتاکردنی بانک تەواوبووە پێش نوێکردنەوە

### 🔒 کۆنترۆڵ
- **شوێنپێی پشکنین**: بەڵگەکاری سەرچاوەی نرخەکان بهێڵەرەوە
- **نوێکردنەوەی یەک لەدوای یەک**: ماوەکان جێمەهێڵەرەوە—بە ڕیز نوێبکەرەوە
- **پێداچوونەوە پێش ناردن**: چاوی دووەم هەبێت لەسەر ڕاستکردنەوەی گرنگ

## چارەسەرکردنی کێشەکان

### هیچ هێڵ حیسابنەکراوە

**بپشکنە**:
- ئایا مامەڵەی پارە بیانیت هەیە؟
- ئایا نرخی ئاڵوگۆڕ تۆمارکراوە بۆ ڕێکەوتی نوێکردنەوە؟
- ئایا نرخی ئێستا جیاوازە لە نرخی تاریخی؟

### هەڵەی "ناتوانرێت بنێردرێت"

**بپشکنە**:
- ئایا بەلایەنی کەمەوە یەک هێڵی نوێکردنەوە هەیە؟
- ئایا هەژماری سەرەکی قازانج/زیانی کۆمپانیاکەت ڕێکخراوە؟
- ئایا ڕێکەوتی نوێکردنەوە لەناو ماوەی ژمێریاری کراوەدایە؟

---

## پرسیارە زۆر پرسراوەکان

**پ: چەند جار دەبێت نوێکردنەوەی نرخ بەڕێوەببەم؟**

و: بەلایەنی کەمەوە، نوێکردنەوە بەڕێوەببە پێش داخستنی هەر ماوەی ژمێریاریەک (مانگانە یان سێمانگانەوە). زۆربەی کۆمپانیاکان مانگانە بەڕێوەیدەبەن بۆ ڕاپۆرتی ناوەندی دروست.

**پ: چی ڕوودەدات کاتێک مامەڵەی پارە بیانی یەکلادەکرێتەوە؟**

و: کاتێک پارە وەردەگریت یان پسووڵەیەک دەدەیت، سیستەم جیاوازی ئاڵوگۆڕی ڕۆشتوو حیساب دەکات. ڕاستکردنەوەی نەڕۆشتوو لە نوێکردنەوە بە کاریگەری پێچەوانەدەکرێتەوە وەک بەشێک لە یەکلاکردنەوەکە.
