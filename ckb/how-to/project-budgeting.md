# تێگەیشتن لە بودجەی پڕۆژە

ئەم ڕێبەرە باس لە **بودجەی پڕۆژە** دەکات — چۆن پلانی تێچووەکانی پڕۆژە دابنێیت، چاودێری خەرجی ڕاستەقینە بکەیت، و پڕۆژەکانت بە قازانجی بهێڵیتەوە. نووسراوە بۆ بەڕێوەبەرانی پڕۆژە، ژمێریاران و خاوەن کارەکان، کە باس لە دروستکردنی بودجە، شیکاری جیاوازی، و ستراتیژییەکانی کۆنترۆڵکردنی تێچوو دەکات.

---

## بودجەی پڕۆژە چییە؟

وێنەی ئەوە بکە خەریکی دروستکردنی خانووێکیت بۆ کڕیارێک کە ڕازی بووە $100,000 بدات. دەبێت بزانیت:
- چەند دەتوانیت بۆ کەرەستە خەرج بکەیت؟
- چەند بۆ دەستی کار؟
- ئایا هیچ بۆ قازانج دەمێنێتەوە؟

**بودجەی پڕۆژە** وەڵامی ئەم پرسیارانە دەداتەوە بە پلاندانانی تێچووەکان **پێش** ئەوەی ڕووبدەن و چاودێریکردنیان **لە کاتی** ڕوودانیاندا.

**بە زمانی بزنس:**
بودجەی پڕۆژە پرۆسەی خەمڵاندنی تێچووەکان، تەرخانکردنی سەرچاوەکان، و چاودێریکردنی خەرجییە ڕاستەقینەکانە بەرامبەر بودجەکە بە درێژایی تەمەنی پڕۆژەکە.

**بۆچی گرنگە؟**
- **ڕێگری لە سوپرایز دەکات**: بزانە ئەگەر لە بودجە دەردەچیت پێش ئەوەی درەنگ بێت
- **قازانج دەپارێزێت**: دڵنیابە لەوەی پڕۆژەکە پارە پەیدا دەکات، نەک لەدەستی بدات
- **کۆنترۆڵ دابین دەکات**: هەنگاو بنێ کاتێک تێچووەکان لە پێشبینییەکان زیاترن
- **خەمڵاندن باشتر دەکات**: فێربوون لە پڕۆژەکانی ڕابردوو بۆ پلاندانانی باشتر

---

## پەیکەری بودجە

لە Kezi ERP، بودجەکان ڕێکخراون بەپێی **بابەت** (category) (کە پێی دەوترێت کۆدی تێچوو):

### بابەتە باوەکانی بودجە

| بابەت | چی دەگرێتەوە | نموونە |
|----------|----------------|---------------|
| **دەستی کار (Labor)** | تێچووی کاتی فەرمانبەر | مووچەی کاتژمێری، مووچەی مانگانە |
| **کەرەستەکان (Materials)** | کاڵای فیزیایی | تەختە، ئاسن، بۆیە، پارچە |
| **ئامێرەکان (Equipment)** | ئامراز و مەکینە | کرێی بەکرێگرتن، داخوران |
| **بەڵێندەرەکان (Subcontractors)** | کاری دەرەکی | پسپۆڕان، بەڵێندەران |
| **گەشتکردن (Travel)** | گەشتی بزنس | فڕۆکە، هۆتێل، مەسافە |
| **سەربار (Overhead)** | تێچووی ناڕاستەوخۆ | بیمە، مۆڵەت، خزمەتگوزاری |

---

## دروستکردنی بودجەی پڕۆژە

### هەنگاوی ١: دەستپێکردنی دروستکردنی بودجە

1. بڕۆ بۆ **بەڕێوەبردنی پڕۆژە ← بودجەکان ← دروستکردن**
2. ئەو **پڕۆژەیە** هەڵبژێرە کە بودجەی بۆ دادەنێیت
3. **ناوی بودجە** بنووسە (نموونە: "بودجەی سەرەتایی" یان "بودجەی پێداچوونەوەکراو ٠٢")

### هەنگاوی ٢: دیاریکردنی دێڕەکانی بودجە

دێڕێک زیاد بکە بۆ هەر بابەتێکی تێچوو:

| بابەت | وەسف | بڕی بودجە |
|----------|-------------|-----------------|
| دەستی کار | کاتژمێری تیمی پەرەپێدەر | $15,000 |
| کەرەستەکان | مۆڵەتی سۆفتوێر | $2,000 |
| ئامێرەکان | کرێی سێرڤەر | $1,000 |
| بەڵێندەرەکان | دیزاینەری UI سەربەخۆ | $3,000 |
| گەشتکردن | سەردانی شوێنی کڕیار | $500 |
| **کۆ** | | **$21,500** |

**نەسیحەت:** دێڕێکی یەدەگ (contingency) زیاد بکە (بەزۆری ٥-١٥٪ی کۆ) بۆ تێچووی چاوەڕواننەکراو.

### هەنگاوی ٣: پێداچوونەوە و پاشەکەوتکردن

پێش کۆتایی هێنان:
- ئایا کۆی بودجەکە لەگەڵ ڕێککەوتنی کڕیار دەگونجێت؟
- ئایا هەموو جۆرە تێچووە چاوەڕوانکراوەکان داپۆشراون؟
- ئایا یەدەگی گونجاو هەیە؟

---

## تێگەیشتن لە بودجە بەرامبەر ڕاستەقینە

هێزی ڕاستەقینەی بودجە لە بەراوردkarیدایە:

```
    ┌─────────────────────────────────────────────────┐
    │               داشبۆردی بودجەی پڕۆژە             │
    │                                                 │
    │  بابەت        │ بودجە     │ ڕاستەقینە │ جیاوازی │
    │──────────────────────────────────────────────── │
    │  دەستی کار    │ $15,000   │ $12,500  │ +$2,500 │ ✅
    │  کەرەستە      │ $2,000    │ $2,400   │ -$400   │ ⚠️
    │  ئامێر        │ $1,000    │ $800     │ +$200   │ ✅
    │  بەڵێندەر     │ $3,000    │ $3,500   │ -$500   │ ⚠️
    │  گەشت         │ $500      │ $200     │ +$300   │ ✅
    │──────────────────────────────────────────────── │
    │  کۆ           │ $21,500   │ $19,400  │ +$2,100 │ ✅
    │                                                 │
    └─────────────────────────────────────────────────┘
    
    کۆی بەکارهێنان: 90.2%
```

### پێوەرە سەرەکییەکان

**جیاوازی (Variance)**
- **موجەب (+)**: لەژێر بودجە (باشە! 🎉)
- **سالب (-)**: لەسەر بودجە (لێکۆڵینەوە بکە! ⚠️)

**ڕێژەی بەکارهێنان %**
- **لەژێر ٨٠٪**: زۆر لەژێر بودجە — هەواڵی باشە یان کەم بەکارهێنان؟
- **٨٠-١٠٠٪**: لەسەر ڕێچکە — پڕۆژەی تەندروست
- **سەروو ١٠٠٪**: لەسەر بودجە — هەنگاو بنێ!

---

## داتای تێچووی ڕاستەقینە لە کوێوە دێت؟

جوانی Kezi ERP ئەوەیە کە تێچووە ڕاستەقینەکان بە ئۆتۆماتیکی لە مۆدیولەکانی ترەوە دێن:

### تێچووی دەستی کار (تایم-شیت)
کاتێک تایم-شیتێک پەسەند دەکرێت:
1. کاتژمێرەکان لێکدانی نرخی تێچووی فەرمانبەر دەکرێت
2. ئەنجامەکە زیاد دەکرێت بۆ "دەستی کار"ی بودجە

**نموونە:**
- ئەحمەد ٢٠ کاتژمێر کاری کردووە
- نرخی تێچووی $75/کاتژمێر
- ڕاستەقینەی دەستی کار زیاد دەکات بە: $1,500

### تێچووی کەرەستە/تری (پسوڵەی دابینکەر)
کاتێک پسوڵەی دابینکەر نیشانەی پڕۆژەی لێ دەدرێت:
1. دێڕەکان دەچنە بابەتی بودجەی پەیوەندیدار
2. بەپێی نەخشەی هەژماری خەرجی

**نموونە:**
- پسوڵەی دابینکەر بۆ "کەرەستەی بۆیە" = $500
- هەژماری خەرجی = کەرەستەکان
- ڕاستەقینەی کەرەستە زیاد دەکات بە: $500

### قەیدی ڕاستەوخۆ
هەر قەیدێک کە نیشانەی پڕۆژەی لێ بدرێت دەچێتە ناو ڕاستەقینەی بودجە بەپێی بابەتی هەژمارەکان.

---

## چاودێریکردنی ڕاستەوخۆ

### ویجێتی جیاوازی بودجە
داشبۆردی بەڕێوەبردنی پڕۆژە چارتێکی بینراو لەخۆدەگرێت کە بەراوردی بودجە و ڕاستەقینە دەکات بۆ هەموو پڕۆژە چالاکەکان.

**ڕەنگەکان تەندروستی نیشان دەدەن:**
- 🟢 **سەوز**: لەژێر بودجە یان لەسەر ڕێچکە
- 🟡 **زەرد**: ٩٠-١٠٠٪ی بودجە بەکارهاتووە
- 🔴 **سور**: لەسەر بودجە

---

## تەواوکاری بۆ ژمێریاران (GL Integration)

بودجەی پڕۆژە تەواوکاری دەکات لەگەڵ دەفتەری گشتی (GL):

### نەخشەی هەژماری خەرجی
هەر بابەتێکی بودجە دەبەسترێتەوە بە هەژماری خەرجی GL:
- دەستی کار → ٥١٠٠ - مووچە و کرێ
- کەرەستەکان → ٥٢٠٠ - تێچووی کەرەستە
- ئامێرەکان → ٥٣٠٠ - کرێی ئامێر
- هتد.

### ڕێڕەوی قەید
کاتێک تێچووەکان تۆمار دەکرێن:
1. Debit: هەژماری خەرجی (بەپێی بابەت)
2. Credit: هەژمارە شایستەکان / بانک

نیشانەی پڕۆژە لەسەر قەیدەکە دڵنیایی دەدات کە دەچێتە ناو ڕاستەقینەی بودجە.

---

## پوختەی باشترین پراکتیزەکان

✅ **واقیعی بە لە خەمڵاندنەکان** — گەشبینی دۆستت نییە لە بودجەدانان

✅ **یەدەگ (Contingency) دابنێ** — ٥-١٥٪ بۆ کێشە چاوەڕواننەکراوەکان

✅ **هەفتانە پێداچوونەوە بکە** — مەهێڵە زیادەڕەوییەکان تووشی شۆکت بکەن

✅ **گۆڕانکارییەکان دۆکیۆمێنت بکە** — هەر گۆڕانکارییەک لە مەودا = کاریگەری بودجە

✅ **لە مێژوو فێر بە** — بەراوردی بودجە و ڕاستەقینە بکە دوای هەر پڕۆژەیەک

✅ **زوو هەنگاو بنێ** — چارەسەرکردنی $500 جیاوازی ئاسانترە لە $5,000
