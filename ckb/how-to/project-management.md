# تێگەیشتن لە بەڕێوەبردنی پڕۆژە

ئەم ڕێبەرە باس لە **بەڕێوەبردنی پڕۆژە** دەکات لە Kezi ERP — چۆن پڕۆژە دروست بکەیت، چاودێری ئەرکەکان بکەیت، بودجە بەڕێوەببەیت، و پسوڵە بۆ کڕیارەکانت بکەیت. نووسراوە بۆ هەمووان (تەنانەت ئەگەر بەڕێوەبردنی پڕۆژە بەلاتەوە قورس بێت!)، باس لە سوڕی ژیانی تەواو دەکات لە دروستکردنی پڕۆژەوە تا کۆتا پسوڵەکردن.

---

## بەڕێوەبردنی پڕۆژە چییە؟

وێنەی ئەوە بکە تۆ کۆمپانیایەکی بیناسازیت. کڕیارێک دەیەوێت ئۆفیسێکی بۆ دروست بکەیت. پێویستە:
- چاودێری هەموو کاتژمێرێک بکەیت کە کرێکارەکانت لەسەر کارەکە بەسەری دەبەن
- چاودێری تێچووی کەرەستە و ئامێرەکان بکەیت
- لە چوارچێوەی ئەو بودجەیە بمێنیتەوە کە لەسەری ڕێککەوتوون
- پسوڵە بۆ کڕیار بکەیت بۆ کاتەکەت و خەرجییەکان

ئەمە ئەوەیە کە **بەڕێوەبردنی پڕۆژە** یارمەیت دەدات بیکەیت — کارە ڕۆژانەکانت دەبەستێتەوە بە سیستەمە ژمێریارییەکەتەوە، بۆیە هەمیشە دەزانیت ئایا پڕۆژەیەک قازانج دەکات!

**بە زمانی بزنس:**
بەڕێوەبردنی پڕۆژە بریتییە لە پرۆسەی پلاندانان، جێبەجێکردن، و چاودێریکردنی کار بۆ بەدەستهێنانی ئامانجە دیاریکراوەکان لە چوارچێوەی کات و بودجەدا. سیستەمەکەمان ئەمە دەبەستێتەوە بە داتای دارایی بۆ بینینی تەواو.

**توخمە سەرەکییەکان:**
1. **پڕۆژەکان**: چوارچێوەی هەموو کارەکە (نموونە: "بینای ئۆفیس بۆ کۆمپانیای ABC")
2. **ئەرکەکان (Tasks)**: پارچە دیاریکراوەکانی کار لە ناو پڕۆژەکەدا (نموونە: "ئەساس"، "کارەبا"، "فنشینگ")
3. **تایم-شیت (Timesheets)**: تۆماری کاتژمێرە کارکراوەکان لەلایەن فەرمانبەرانەوە
4. **بودجەکان**: تێچووی پلان بۆ دانراو بەرامبەر خەرجی ڕاستەقینە
5. **پسوڵەکان**: پارە وەرگرتن لە کڕیار بەپێی کارە تەواو کراوەکان

---

## چۆن کاردەکات لە Kezi ERP

سیستەمەکەمان چارەسەرێکی تەواوی بەڕێوەبردنی سوڕی ژیانی پڕۆژە دابین دەکات کە تەواوکاری کراوە لەگەڵ ژمێریاری.

### ١. دروستکردنی پڕۆژە

وەک کردنەوەی فایلێکی کاری نوێ بیهێنە پێش چاوت. تۆ ئەمانە دیاری دەکەیت:
- **ناوی پڕۆژە**: خەریکی چی دەبیت؟
- **کڕیار**: بۆ کێی دەکەیت؟
- **بەرواری دەستپێک/کۆتایی**: کار کەی دەست پێدەکات و کەی تەواو دەبێت؟
- **جۆری پسوڵەکردن**: چۆن پارە وەردەگریت؟ (نرخی جێگیر، کات و کەرەستە، یان بێ بەرامبەر)
- **بودجە**: پێشبینی دەکەیت چەند خەرج بکەیت؟

**نموونە:**
"دیزاینکردنەوەی وێبسایت بۆ XYZ Corp" بۆ کڕیار XYZ Corp، دەستپێک ١ی کانوونی دووەم، بە کاتژمێر هەژمار دەکرێت.

**لە کوێ دەیدۆزیتەوە:** بەڕێوەبردنی پڕۆژە ← پڕۆژەکان ← Create

---

## دۆخەکانی پڕۆژە: سوڕی ژیان

هەر پڕۆژەیەک بەم قۆناغانەدا تێدەپەڕێت:

```
    ┌─────────────────────────────────┐
    │   ڕەشنووس (DRAFT)               │
    │   پڕۆژە پلانی بۆ دادەنرێت       │
    └──────────────┬──────────────────┘
                   │ چالاککردن
                   ▼
    ┌─────────────────────────────────┐
    │   لە کارکردن (IN PROGRESS)      │
    │   کار بە چالاکی بەڕێوەدەچێت     │
    └──────────────┬──────────────────┘
                   │ تەواوکردن
                   ▼
    ┌─────────────────────────────────┐
    │   تەواو بوو (COMPLETED)         │
    │   هەموو کارەکان تەواو بوون      │
    └──────────────┬──────────────────┘
                   │ (ئیختیاری)
                   ▼
    ┌─────────────────────────────────┐
    │   ڕەتکراوە (CANCELLED)          │
    │   پڕۆژە زوو وەستێنرا            │
    └─────────────────────────────────┘
```

**یاسا گرنگەکان:**
- تەنها دەتوانیت تایم-شیت تۆمار بکەیت بۆ پڕۆژەی **لە کارکردن**
- تەنها پڕۆژەی **تەواو بوو** دەتوانێت پسوڵەی کۆتایی وەربگرێت
- پڕۆژەی **ڕەتکراوە** هەموو چالاکییەکان دەوەستێنێت

---

## ئەرکەکان: دابەشکردنی کارەکە

بەڕێوەبردنی پڕۆژەی گەورە قورسە. ئەرکەکان یارمەیت دەدەن کارەکە ڕێکبخەیت بۆ پارچەی بەڕێوەبراو.

### دروستکردنی ئەرکەکان

بۆ پڕۆژەی "دیزاینکردنەوەی وێبسایت"ەکەمان:

| ناوی ئەرک | کاتژمێری خەمڵێنراو | راسپێردراوە بە |
|-----------|-----------------|-------------|
| دۆزینەوە و پلاندانان | ٢٠ | سارا |
| دیزاینی UI/UX | ٤٠ | ئەحمەد |
| پەرەپێدان | ٨٠ | تیم |
| تاقیکردنەوە و QA | ٢٠ | لەیلا |
| بڵاوکردنەوە و ڕاهێنان | ١٠ | هەمووان |

### دۆخەکانی ئەرک

هەر ئەرکێک سوڕی ژیانی خۆی هەیە:
- **بۆ کردن (To Do)**: هێشتا دەستی پێ نەکراوە
- **لە کارکردن (In Progress)**: کار بەڕێوەدەچێت
- **تەواو بوو (Completed)**: ئەرک تەواو بووە
- **ڕەتکراوە (Cancelled)**: ئەرک لابراوە

---

## تایم-شیت: چاودێریکردنی کاتژمێرەکانی فەرمانبەر

تایم-شیت وەڵامی ئەم پرسیارە دەداتەوە: **"کێ کاری لەسەر چی کرد، و بۆ ماوەی چەند؟"**

### ڕێڕەوی کاری تایم-شیت

```
    ┌─────────────────────────────────┐
    │   ڕەشنووس                       │
    │   فەرمانبەر کاتژمێر داخڵ دەکات  │
    └──────────────┬──────────────────┘
                   │ ناردن
                   ▼
    ┌─────────────────────────────────┐
    │   نێردراو                       │
    │   چاوەڕێی پەسەندکردن            │
    └──────────────┬──────────────────┘
                   │
        ┌──────────┴──────────┐
        │                     │
        ▼ پەسەندکردن          ▼ ڕەتکردنەوە
    ┌───────────┐         ┌───────────┐
    │ پەسەندکراو│         │ ڕەتکراوە  │
    │ کاتژمێر   │         │ فەرمانبەر │
    │ هەژمارکرا!│         │دەبێت چاکی کات│
    └───────────┘         └───────────┘
```

## بودجەی پڕۆژە: مانەوە لەسەر ڕێچکە

بودجەکان یارمەیت دەدەن وەڵامی ئەمە بدەیتەوە: **"ئایا پارە پەیدا دەکەین یان لەدەستی دەدەین؟"**

### چاودێریکردنی بودجە

سیستەمەکە بە شێوەی ئۆتۆماتیکی چاودێری دەکات:
- **بڕی بودجە**: ئەوەی پلانت بۆ داناوە خەرجی بکەیت
- **بڕی ڕاستەقینە**: ئەوەی بە ڕاستی خەرجت کردووە (لە تایم-شیتە پەسەندکراوەکان، پسوڵەی دابینکەر، هتد)
- **جیاوازی**: جیاوازییەکە (موجەب = لەژێر بودجە ✅، سالب = لەسەر بودجە ⚠️)
- **ڕێژەی بەکارهێنان %**: چەند لە بودجەکە بەکارهاتووە

**نوێبوونەوەی ڕاستەوخۆ:** هەر کاتێک تایم-شیتێک پەسەند دەکرێت یان خەرجییەک تۆمار دەکرێت، داشبۆردی بودجە بە ئۆتۆماتیکی نوێ دەبێتەوە!

---

## پسوڵەی پڕۆژە: وەرگرتنی پارە

لە کۆتاییدا، ئامانجی هەموو ئەم چاودێریکردنە — پارە وەرگرتن لە کڕیارەکەت!

### جۆرەکانی پسوڵەکردن

**١. کات و کەرەستە (Time & Materials)**
- پسوڵە بۆ کڕیار بەپێی کاتژمێری کارکراوی ڕاستەقینە
- هەر کاتژمێرێک نرخێکی هەیە (نموونە: $150/کاتژمێر بۆ پەرەپێدەران)
- باشە بۆ: ڕاوێژکاری، پەرەپێدان، کاری داهێنەرانە

**٢. نرخی جێگیر (Fixed Price)**
- پسوڵەکردنی بڕێکی دیاریکراو بەبێ گوێدانە کاتژمێرەکان
- وتووتە $20,000، هەر $20,000 وەردەگریت
- باشە بۆ: بیناسازی، پڕۆژەی دیاریکراو

**٣. بێ بەرامبەر (Non-Billable)**
- پڕۆژەی ناوخۆیی بەبێ پسوڵەی کڕیار
- ڕاهێنان، توێژینەوە و پەرەپێدان (R&D)، ئامرازی ناوخۆیی

---

## باشترین پراکتیزەکان

### ١. هەمیشە ئەرکەکان بەکاربهێنە
کات ڕاستەوخۆ بۆ پڕۆژەکان تۆمار مەکە — دابەشی بکە بۆ ئەرکەکان. ئەمە داتای باشترت دەداتێ کە کات لە کوێ خەرج دەکرێت.

### ٢. تایم-شیت هەفتانە بنێرە
چاوەڕێی کۆتایی مانگ مەکە. ناردنی هەفتانە داتا بە تازەیی دەهێڵێتەوە و هەڵەکان زوو دەگرێت.

### ٣. هەفتانە پێداچوونەوەی جیاوازی بودجە بکە
داشبۆردی بودجە هەموو هەفتەیەک سەیر بکە. پێزانینی زوو بە زیادەڕەوی خەرجی کاتت دەداتێ بۆ ڕاستکردنەوە.

### ٤. پڕۆژە تەواو بووەکان دابخە
کاتێک بە تەواوی پسوڵە کرا، پڕۆژەکان وەک **تەواو بوو** دیاری بکە. ئەمە ڕێگری دەکات لە داخڵکردنی تایم-شیتی هەڵە.
