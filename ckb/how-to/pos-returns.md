ئەم ڕێنماییە ڕوونی دەکاتەوە چۆنیەتی بەڕێوەبردنی گەڕاندنەوەی کاڵاکان لە کاشێری فرۆشگا، لەوانەش ڕووکاری پەسەندکردن و چۆنیەتی یەکخستنی گەڕاندنەوەکان لەگەڵ کۆگا و ژمێریاری.

---

## گەڕاندنەوەی کاشێر چییە؟

گەڕاندنەوەی کاشێر پرۆسەی گەڕاندنەوەی کاڵاکان لەلایەن کڕیار دژ بە فرۆشێکی پێشتری تەواوکراوەکە تۆمار دەکات. ئەمەش مەرجیەکانی خوارەوەی دەگرێتەوە:

- **گەڕاندنەوەی پارە** بۆ کڕیار (نەقد، بانک، یان شێوازی پارەدانی ئەسڵی)
- **گەڕاندن بۆ کۆگا** (گەڕاندنەوەی کاڵاکان بۆ کۆگا)
- **تۆماری ژمێریاری** (نوسراوی کرێدیت + گەڕاندنەوەی پارەدان)

---

## لەکوێ دەتوانی لەناو UI بیبینیت

بچۆ بۆ **کاشێر ← کارەکان ← گەڕاندنەوەکان**

## بارودۆخەکانی گەڕاندنەوە

| بارودۆخ | ڕوونکردنەوە |
|---------|------------|
| **ڕەشنووس** | گەڕاندنەوەکە دروستکراوە بەڵام هێشتا نەردراوە |
| **چاوەڕوانی پەسەندکردن** | گەڕاندنەوەکە کاتژمێری پەسەندکردنی بەڕێوەبەر دەچاوەڕووێت |
| **پەسەندکراو** | گەڕاندنەوەکە پەسەندکراوە و ئامادەیە بۆ کارپێکردن |
| **لە پرۆسەدایە** | گەڕاندنەوەکە لە کارپێکردندایە |
| **تەواوکراو** | گەڕاندنەوەکە بە تەواوی کارپێکراوە، پارەکە گەڕاوەتەوە و کۆگا نوێکراوەتەوە |
| **ڕەتکراوە** | گەڕاندنەوەکە لەلایەن بەڕێوەبەرەوە ڕەتکراوە |
| **هەڵوەشێنراوە** | گەڕاندنەوەکە پێش کارپێکردن هەڵوەشێنراوە |

---

## ڕووکاری گەڕاندنەوە

```
ڕەشنووس → چاوەڕوانی پەسەندکردن → پەسەندکراو → تەواوکراو
                                   ↘ ڕەتکراوە
```

1. **دروستکرا** لە کاشێری POS یان باکەند.
2. **نەردرا** بۆ پەسەندکردنی بەڕێوەبەر (ئەگەر سیاسەتی گەڕاندنەوە داوای دەکات).
3. **پەسەندکرا** (یان ڕەتکرا) لەلایەن بەکارهێنەرێکی مەجازەوە.
4. **کارپێکرا** — بە خۆکاری دروستی دەکات:
   - **نوسراوی کرێدیت** (فاکتووری نەرێنی) لە مۆدیولی ژمێریاری
   - **گەڕاندنەوەی پارەدان** (پارەدانی ئەرزانی دەرەوەی گەڕاوەتەوە)
   - **جووڵەی کۆگا** بۆ گەڕاندنەوەی کاڵای گەڕاوەتەوە بۆ کۆگا (ئەگەر پێویست)

---

## وردەکارییەکانی گەڕاندنەوە

هەر گەڕاندنەوەی کاشێر ئەمانەی خوارەوەی دەگرێتەوە:

### زانیاری سەرەکی
- **ژمارەی گەڕاندنەوە**: ناسنامەی تایبەت (بۆ نموونە `RET-0001`)
- **داواکاری ئەسڵی**: داواکاری POS-ی گەڕاندنەوەکە سەرچاوەی دەگرێت
- **بەرواری گەڕاندنەوە**: کاتی دروستکردنی گەڕاندنەوه
- **بارودۆخ**: قۆناغی ئێستای ڕووکار
- **بڕی گەڕاوەتەوەی پارە**: کۆی گشتی بڕی گەڕاوەتەوە
- **شێوازی گەڕاوەتەوەی پارە**: چۆن کڕیار گەڕاوەتەوەی پارەی وەردەگرێت

### هێڵەکانی گەڕاندنەوە
هەر هێڵێک بەرامبەر کاڵایەکی گەڕاوەتەوەیە:
- **کاڵا**
- **بڕی گەڕاوەتەوە**
- **نرخی تاک** (لە داواکاری ئەسڵی)
- **بڕی گەڕاوەتەوەی هێڵ**
- **بارودۆخ**: باش، کراوەیە، زیانمەند، کەموکوڕ
- **گەڕاندن بۆ کۆگا**: ئایا کاڵاکە دەگەڕێتەوە بۆ کۆگا

### یەکخستن لەگەڵ ژمێریاری
کاتێک کارپێکرا، گەڕاندنەوەکە بەستراوەتەوە بە:
- **نوسراوی کرێدیت** — فاکتوویەکی نەرێنی کە فرۆشتنی ئەسڵی دەگەڕێنێتەوە
- **گەڕاندنەوەی پارەدان** — پارەدانی دەرچووی دەرەوە کە گەڕاوەتەوەی پارەکە پشتڕاست دەکات
- **جووڵەی کۆگا** — ئامادەکاری گەڕاندنی کاڵاکان بۆ کۆگا تۆمار دەکات

---

## کردەکان

لە لاپەڕەی **بینین** ی گەڕاندنەوەیەک، بەکارهێنەرانی مەجاز دەتوانن:

### پەسەندکردنی گەڕاندنەوە
- بەردەستە کاتێک بارودۆخ **چاوەڕوانی پەسەندکردن**ە
- کاربەکاری پەسەندکەر و کاتەکەی تۆمار دەکات
- بارودۆخ دەگوازێتەوە بۆ **پەسەندکراو**

### ڕەتکردنەوەی گەڕاندنەوە
- بەردەستە کاتێک بارودۆخ **چاوەڕوانی پەسەندکردن**ە
- داوای هۆکاری ڕەتکردنەوە دەکات
- هۆکارەکە زیاددەکرێت بۆ تێبینییەکانی گەڕاندنەوە

### کارپێکردنی گەڕاندنەوە
- بەردەستە کاتێک بارودۆخ **پەسەندکراو**ە
- پرۆسەی کارپێکردنی تەواو دەستپێدەکات:
  1. نوسراوی کرێدیت دروستدەکات (یەکسەر دانراودەرچووە)
  2. گەڕاندنەوەی پارەدان دروستدەکات (یەکسەر پشتڕاستکراوە)
  3. جووڵەی کۆگا دروستدەکات بۆ کاڵاکانی کە بۆ گەڕاندن بۆ کۆگا نیشانکراون
